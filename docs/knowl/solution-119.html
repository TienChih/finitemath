Solution 6.4.4.1.

The numerator has degree 2 and denominator has degree 3, so the long term behavior can be approximated by \(f(x)\approx\frac{x^2}{x^3}\approx \frac{1}{x}\) and so the long term behavior approximates \(y=0\text{.}\)

The roots of the numerator are \(-1, 5\) so \(f(-1)=0=f(5)\text{.}\) These both have multiplicty 1, which is odd, so the function changes sign at these values.

The roots of the numerator are \(-2, 2\) so we have vertical asymptotes \(x=-2, x=2\text{.}\) The root \(x=-2\) has multiplicity 2, which is even, so the function does not change signs here. The root \(x=2\) has multiplicty 1, which is odd, so the function changes sign at this value.
