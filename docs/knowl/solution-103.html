Solution 7.6.2.1.

We note that:

\begin{align*} R'(t)\amp = \frac{d}{dt}[10-8e^{-0.02t}]\\ \amp = 0-8e^{-0.02t}\frac{d}{dt}[-0.02t]\\ \amp = -8e^{-0.02t}(-0.02)\\ \amp = 0.16e^{-0.02t}\text{.} \end{align*}

Thus \(R'(30)=0.16e^{-0.02(30)}\approx 0.088\) so increasing by about \(\$88 \) a day.
