We can either sum together all the ways to pick some number of these moves to be “rights” and the rest “ups”. Or we could take out 10 moves, and count the ways to select “up” or “right” for each. Either way we obtain

\begin{equation*} \sum_{k=0}^{10} {10\choose k}=2^{10}=1024 \end{equation*}

paths of length 10.
