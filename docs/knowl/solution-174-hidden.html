Let \(I\) denote “infected” and \(T\) denote “tested positive”. We are given \(P(T|I)=0.9, P(T|I^c)=0.08, P(T)=0.3.\) We are asked to find \(P(I|T)\text{.}\) We first find \(P(I)\text{:}\)

\begin{align*} P(T)\amp =P(T|I)P(I)+P(T|I^c)(1-P(I))\\ P(I) \amp =\frac{P(T)-P(T|I^c)}{P(T|I)-P(T|I^c)}\\ \amp =\frac{0.3-0.08}{0.9-0.08}=\frac{0.22}{0.82}=\frac{11}{41}\approx 0.26829 \end{align*}

Which we can see: Moreover, we now know \(P(I)=\frac{11}{41}, P(T)=0.3, P(T\cap I)=P(T|I)P(I)=(0.9)\frac{11}{41}=\frac{99}{410}\) So we can fill out the following venn diagram:

So \(P(I|T)=\frac{P(I\cap T)}{P(T)}=\frac{99/410}{0.3}=\frac{33}{41}\approx 0.80488\text{.}\)
