We've already seen some examples of these.

  • A constant linear function \(y=c\) is a polynomial of degree 0, since \(y=cx^0\text{.}\)
  • A constant linear function of the form \(y=mx+b\) with non-zero slope is a polynomial of degree 1, since \(y=mx^1+bx^0\text{.}\)
  • A constant quadratic function \(y=ax^2+bx+c\) is a polynomial of degree 2, since \(y=ax^2+bx^1+cx^0\text{.}\)
