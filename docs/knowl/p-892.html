We should also note that there is a real kinship between exponential functions and linear functions. Linear functions have the form \(\ell(x)=mx+b\text{,}\) which we can think of as

\begin{equation*} \ell(x)=\overbrace{m+m+\cdots+m}^x+b\text{,} \end{equation*}

with \(x\) copies of \(m\text{.}\) Similarly we can see that

\begin{equation*} f(x)=b\cdot a^x=\overbrace{a\cdot a\cdots a}^x\cdot b\text{,} \end{equation*}

so it is really the multiplicative analogue of the linear function, where the \(b\)'s are the initial value of the function when \(x=0\) and the growth rate of the function is totally determined by \(a\) (whereas it's determined by \(m\) for linear functions).
