Solution 1.3.8.1.

We start by finding the slope of the second line. As usual:

\begin{equation*} m=\frac{\Delta y}{\Delta x} =\frac{9-1}{5-1}=\frac{8}{4}=2. \end{equation*}

Since our line is parallel to this one, they share the same slope of \(m=2\text{.}\) Knowing that \we pass through \((2,-1)\text{,}\) we can solve for a \(y-\)intercept:

\begin{align*} y\amp = mx+b\\ -1\amp = 2(2)+b\\ b\amp = -1-4=-5\\ y\amp = 2x-5. \end{align*}

Or, more directly:

\begin{align*} y-y_0\amp = m(x-x_0)\\ y-1\amp = 2(x-2)\\ y-1\amp = 2x-4\\ y\amp = 2x-5. \end{align*}
