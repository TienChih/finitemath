The numerator and denominator are both degree 2, so the long term behavior can be approximated by \(f(x)\approx\frac{x^2}{x^2}\approx 1\) and so there is a horizontal asymptote \(y=1\text{.}\)

The roots of the numerator are \(-2, 8\) so \(f(-2)=0=f(8)\text{.}\) These both have multiplicty 1, which is odd, so the function changes sign at these values.

The roots of the numerator are \(-3, 2\) so we have vertical asymptotes \(x=-3, x=2\text{.}\) These both have multiplicty 1, which is odd, so the function changes sign at these values.
