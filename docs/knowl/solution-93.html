Solution 7.4.6.1.

We have:

  1. \(C'(x)=\frac{d}{dx}[2x]=2\text{.}\) So \(\$2\) per product
  2. \(R'(x)=\frac{d}{dx}[6x-\frac{x^2}{100}]=6-\frac{x}{50}\text{.}\) So \(\$(6-\frac{x}{50})\) per product
  3. \(P(x)=R(x)-C(x)=6x-\frac{x^2}{100}-2x=4x-\frac{x^2}{100}.\text{.}\)
  4. Since \(P'(x)=\frac{d}{dx}[4x-\frac{x^2}{100}]=4-\frac{x}{50} \text{,}\) we note that this is zero when:
    \begin{align*} 4-\frac{x}{50}\amp =0\\ 4\amp= \frac{x}{50}\\ x\amp=200\text{.} \end{align*}
    So marginal profit is 0 when 200 units of the product are produced.
  5. We have \(P(200)=4(200)-\frac{200^2}{100} =\$400\text{.}\)
