Solution 9.4.11.1.

We note that the device has a \(0.3\) chance to fail in the first year, \((0.7)(0.3)\) chance to fail the second since it had to survive the first, \((0.7)^2(0.3)\) to fail the third year, after which the policy ends. So if \(X\) is the payout, then it has distribution

\begin{equation*} \begin{array}{|c|cccc|} \hline x \amp 3000 \amp 2000 \amp 1000 \amp 0 \\ \hline P(X=x) \amp (0.3) \amp (0.7)(0.3) \amp (0.7)^2(0.3) \amp 1-(0.3)-(0.7)(0.3)-(0.7)^2(0.3)\\ \hline \end{array} \end{equation*}

Which can be rewritten

\begin{equation*} \begin{array}{|c|cccc|} \hline x \amp 3000 \amp 2000 \amp 1000 \amp 0 \\ \hline P(X=x) \amp 0.3 \amp 0.21 \amp 0.147 \amp 0.343\\ \hline \end{array} \end{equation*}

Which has expected value

\begin{equation*} E(x)=3000\cdot0.3+2000\cdot0.21+1000\cdot0.147+0\cdot0.343=1467. \end{equation*}

So an expected payout of \(\$1467\text{:}\)
