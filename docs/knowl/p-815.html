So for \(f(x)=2^x\text{,}\) each time we increase \(x\) by 1, the value of \(f(x)\) doubles, and each time we decrease \(x\) by 1, the value of \(>f(x)\) is halved.

\begin{equation*} \begin{array}{|c||c|c|c|c|c|c|} \hline x \amp 0\amp 1\amp 2\amp 3\amp -1 \amp -2\\ \hline f(x)\amp2^0=1\amp 2^1=2\amp 2^2=4\amp 2^3=8\amp 2^{-1}=\frac{1}{2} \amp 2^{-2}=\frac{1}{4}\\ \hline \end{array} \end{equation*}
