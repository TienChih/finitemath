Solution 7.4.7.1.

We note that revenue is price times quantity demanded so \(R(q)=p(q)q=(\frac{500}{q^2}+800)q=\frac{500}{q}+800q.\) From here we can find the marginal revenue: \(R'(q)=\frac{d}{dq}[\frac{500}{q}+800q]=\frac{d}{dq}[500q^{-1}+800q]=-500q^{-2}+800=\frac{-500}{q^2}+800\text{.}\)

Thus, when demand is 4, the marginal revenue is \(R'(4)=\frac{-500}{4^2}+800=\$768.75\) per unit. We can see this below:
