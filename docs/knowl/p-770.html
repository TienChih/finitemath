Let \(P\) denote the present value of a investment/debt. Let \(R\) denote the size of the regular payments per time period, \(i\) denote the interest rate per time period and \(n\) the number of time periods. Then

\begin{equation*} P = R\frac{1-(1+i)^{-n}}{i}. \end{equation*}

Be sure you keep track of what time period (years, days, months etc.) you're using and stay consistent.
