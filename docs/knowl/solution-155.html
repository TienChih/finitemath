Solution 9.4.10.1.

So there are 3 defective and 17 non-defective products.

  1. We note that each selection comes from either defective or non-defective products. So:
    \begin{equation*} \begin{array}{|c|cccc|} \hline x \amp 0 \amp 1 \amp 2 \amp 3 \\ \hline P(X=x) \amp \frac{C(3,0)C(17,5)}{C(20,5)}\amp \frac{C(3,1)C(17,4)}{C(20,5)} \amp \frac{C(3,2)C(17,3)}{C(20,5)} \amp \frac{C(3,3)C(17,2)}{C(20,5)}\\ \hline \end{array} \end{equation*}
    This may be rewritten
    \begin{equation*} \begin{array}{|c|cccc|} \hline x \amp 0 \amp 1 \amp 2 \amp 3 \\ \hline P(X=x) \amp \frac{6188}{15504}\amp \frac{7140}{15504} \amp \frac{2040}{15504} \amp \frac{136}{15504}\\ \hline \end{array} \end{equation*}
  2. This would be computed:
    \begin{equation*} E(X)=0\cdot\frac{6188}{15504}+1\cdot\frac{7140}{15504}+2\cdot\frac{2040}{15504}+3\cdot\frac{136}{15504}=0.75. \end{equation*}
  3. We first note:
    \begin{equation*} E(X^2)=0^2\cdot\frac{6188}{15504}+1^2\cdot\frac{7140}{15504}+2^2\cdot\frac{2040}{15504}+3^2\cdot\frac{136}{15504}=\frac{81}{76}\approx 1.0658. \end{equation*}
    This leads:
    \begin{equation*} Var(X)=E(X^2)-E(X)^2=\frac{81}{76}-0.75^2=\frac{153}{304}\approx0.5033. \end{equation*}
    Finally:
    \begin{equation*} \sigma_X=\sqrt{Var(X)}=\sqrt{\frac{153}{304}}\approx 0.7094. \end{equation*}
