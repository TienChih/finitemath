Solution 9.3.7.1.

Let \(W\) denote “both marbles are white”, \(J_1\) denote “Jar one was chosen”. Note that \(P(J_1)=0.5\text{.}\)

  1. So by Bayes Theorem:
    \begin{align*} P(W)\amp =P(W|J_1)P(J_1)+P(W|J_1^c)(1-P(J_1))\\ P(W)\amp =\frac{C(5,2)}{C(11,2)}(0.5)+\frac{C(8,2)}{C(15,2)}(0.5)\\ P(W)\amp =\frac{10}{55}(0.5)+\frac{28}{105}(0.5)\\ P(W)\amp =\frac{36}{165}\approx 0.224242 \end{align*}
  2. Note, we assume \(W\text{,}\) so this is \(P(J_2|W)\text{.}\) This is then computed:
    \begin{equation*} P(J_2^c|W)=\frac{P(J_2\cap W)}{P(W)}=\frac{P(W|J_1^c)(1-P(J_1))}{P(W)}=\frac{\frac{28}{105}(0.5)}{\frac{36}{165}}=\frac{11}{18}\approx 0.6111111. \end{equation*}
    After all, white marbles are more likely in Jar 2, so an all white selection is more likely to come from Jar 2.
