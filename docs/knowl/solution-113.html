Solution 8.2.22.1.

If we let \(R\) denote “has done undergraduate research” and \(C\) denote “has gone to a conference” then we are given \(P(R)=0.2, P(C)=0.25, P(R\cup C)=0.3\text{:}\)

  1. This is \(P(R\cap C)\text{:}\)
    \begin{align*} P(R\cup C)\amp = P(R)+P(C)-P(R\cap C) \\ 0.3\amp = 0.2+0.25-P(R\cap C) \\ P(R\cap C)\amp=0.15 \end{align*}
    From here we may fill out the following diagram
  2. This is \(P(R\cap C^c)\text{,}\) which we can see is \(P(R\cap C^c)=0.05\text{.}\)
  3. This is \(P(R^c\cap C^c)\text{,}\) which we can see is \(P(R^c\cap C^c)=0.7\text{.}\)
