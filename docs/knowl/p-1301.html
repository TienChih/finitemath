Since the problem is we can scale the size of this sum by increasing the size of the data set, we can reverse that process by dividing by the size of the data set.

\begin{equation*} \frac{(1-3)^2+(2-3)^2+(3-3)^2+(4-3)^2+(5-3)^2}{5}=\frac{10}{5}=2 \end{equation*}
\begin{equation*} \frac{(1-3)^2+(2-3)^2+(3-3)^2+(4-3)^2+(5-3)^2+(1-3)^2+(2-3)^2+(3-3)^2+(4-3)^2+(5-3)^2}{10}=\frac{20}{10}=2 \end{equation*}

and now these data sets are equally “spread out”.
