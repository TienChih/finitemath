Suppose we had

\begin{equation*} M=\begin{pmatrix} 1 \amp 1 \\ -1 \amp -1\end{pmatrix}. \end{equation*}

If we tried to find it's inverse as we have above, we would have:

\begin{equation*} \begin{pmatrix} 1 \amp 1 \\ -1 \amp -1\end{pmatrix}\begin{pmatrix} a \amp b \\ c \amp d\end{pmatrix}=\begin{pmatrix} 1 \amp 0 \\ 0 \amp 1\end{pmatrix}. \end{equation*}

and the system

\begin{align*} a-c\amp=1\\ b-d\amp=0\\ a-c\amp=0\\ b-d\amp=1 \end{align*}

But wait! How can \(a-c=1\) and \(a-c=0\text{?}\) This is a non-solvable system. We can see it here as well:

We see that we have a row \(0a+0b+0c+0d=1\text{,}\) which has no solution.
