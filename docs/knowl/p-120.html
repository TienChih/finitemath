We as usual have options. Knowing that \(m=-1.5\) and that we pass through \((2,5)\text{,}\) we can solve for a \(y-\)intercept:

\begin{align*} f(x)\amp = mx+b\\ 5\amp = -1.5(2)+b\\ b\amp = 5+3=8\\ f(x)\amp = -1.5x+8. \end{align*}

Or, more directly:

\begin{align*} f(x)-y_0\amp = m(x-x_0)\\ f(x)-5\amp = -1.5(x-2)\\ f(x)-5\amp = -1.5x+3\\ f(x)\amp = -1.5x+8. \end{align*}
