Solution 1.4.5.1.

We note that marginal cost is the price per can, i.e. the slope of the cost/can line. We recall that

\begin{equation*} m=\frac{\Delta y}{\Delta x}=\frac{7500-5500}{1500-1000}=\frac{2000}{5000}=4 \end{equation*}

or $4/can.

To find the fixed cost, this is the cost before production, or the cost of producing 0 cans, i.e. the y-intercept:

\begin{align*} C(x)\amp=mx+b\\ C(x)\amp=4x+b\\ 5500\amp=4(1000)+b\\ 5500\amp=4000+b\\ b\amp=1500. \end{align*}

So the fixed cost of $1500 prior to production. Note that we could have found these in other ways as well.
