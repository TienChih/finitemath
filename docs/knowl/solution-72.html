Solution 6.5.2.1.

We note that since we are given \(a=1.2\text{,}\) it suffices to find \(b\text{.}\) Consider that:

\begin{align*} 5=h(2)\amp = b(1.2)^2\\ 1.44b\amp = 5\\ b\amp = \frac{5}{1.44}=\frac{125}{36}\approx 3.47222 \end{align*}

Thus \(h(x)=\frac{125}{36}\cdot (1.2)^x\text{.}\)
