If 70 million tons are produced, then we have \(y=70\text{.}\) So we have:

\begin{align*} y\amp= 1.37993x-24.0358\\ 70\amp= 1.37993x-24.0358\\ 1.37993x\amp=94.0358\\ x\amp=\frac{94.0358}{1.37993}\approx 68.145 \end{align*}

So if 70 million tons of soybeans are produced, it on average would take 68.145 million acres to produce.
