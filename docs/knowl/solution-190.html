Solution 9.2.8.1.

Notice that for any path of length \(n\text{,}\) we can select some of the \(n\) steps to be “rights” and the rest would be “ups”. So paths to \((2,3)\) would correspond to all the way to take 5 moves, and select 2 of them to be “rights”. This would be \({5\choose 2}=10\) paths.
