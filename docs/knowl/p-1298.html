So, given a data set \(1,2,3,4,5\text{,}\) we should all see the mean is 3. We could try to measure the spread of this data set by taking the differences between the mean and the data points and summing them up:

\begin{equation*} (1-3)+(2-3)+(3-3)+(4-3)+(5-3) \end{equation*}

However, when we look at this, we see that the sum of these values is 0. So this, as it stands, is not a great measure of the spread.
