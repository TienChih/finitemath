Let us find both the maximum and minimum solutions of \(g(x,y)=3x+2y\) subject to:

\begin{align*} x+y\amp \leq 8\\ 2x+y\amp \leq 10\\ x\amp \geq 0\\ y\amp \geq 0 \end{align*}

Using Desmos to sketch the feasible region, we obtain: We can observe the corners of this polygon occur at \((0,0), (0,8), (2,6), (5,0)\text{.}\) We could evaluate \(f(x,y)\) at each of these points, but Desmos can be utilized here as well. Thus \(f(x,y)\) achieves a maximum at \((2,6)\) with maximum value \(f(2,6)=3\cdot2+2\cdot6=18\) and is minimized at \((0,0)\) with minimum value \(f(0,0)=3\cdot0+2\cdot0=0\text{.}\)
