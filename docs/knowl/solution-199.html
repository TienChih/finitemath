Solution 9.3.5.1.

There are \(C(20,3)=1140\) ways to pick 3 people out of 20.

  1. There are \(C(5,3)=10\) ways to pick only sick people, so the probability this happens is
    \begin{equation*} \frac{C(5,3)}{C(20,3)}=\frac{10}{1140}\approx 0.0088. \end{equation*}
  2. There are \(C(15,3)=455\) ways to pick only healthy people, so the probability this happens is
    \begin{equation*} \frac{C(15,3)}{C(20,3)}=\frac{455}{1140}\approx 0.3991. \end{equation*}
  3. There are \(C(5,2)\cdot C(15,1)=10\cdot 15=150\) ways to pick two sick and one healthy person, so the probability this happens is
    \begin{equation*} \frac{C(5,2)\cdot C(15,1)}{C(20,3)}=\frac{10\cdot 15}{1140}\approx 0.1316. \end{equation*}
  4. The probability that at least one person tested sick is \(1-\frac{C(15,3)}{C(20,3)}\text{.}\) Thus assuming this happens, the probability everyone tests sick is
    \begin{equation*} \frac{C(5,3)/C(20,3)}{1-C(15,3)/C(20,3)}\approx \frac{0.0088}{0.6009}\approx 0.0146. \end{equation*}
