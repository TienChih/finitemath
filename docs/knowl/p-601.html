In general, if the interest rate per time unit is \(i\text{,}\) then after \(n\) time units:

\begin{equation*} \begin{array}{|c|c|} \hline \text{ Time Unit } \amp \text{ Debt } \\ \hline 0 \amp P\\ 1\amp P+iP=(1+i)P\\ 2\amp (1+i)P+i(1+i)P=(1+i)^2P\\ 3 \amp (1+i)^2P+i(1+i)^2P=(1+i)^3P\\ \vdots \amp \vdots\\ n \amp (1+i)^nP\\ \hline \end{array} \end{equation*}
