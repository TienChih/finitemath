So we have:

\begin{align*} R\amp = 2000\\ S\amp = 50000\\ i\amp = 0.10 \end{align*}

So we are solving:

\begin{align*} 50000\amp = 2000\frac{(1.1)^{n}-1}{.1}\\ 2.5\amp = 1.1^n-1\\ 3.5\amp = 1.1^n\\ \ln(3.5)\amp= n\ln(1.1)\\ n\amp = \frac{\ln(3.5)}{\ln(1.1)}\approx 13.144 \end{align*}

We could also plot \(y=1.1^x\) and set this against \(y=3.5\) to see what \(n\) should be: We can observe that these curves intersect when \(x\approx 13.144\text{.}\)

So each year, Katya will have saved \(\$50000\) in just over 13 years.
