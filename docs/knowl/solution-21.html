Solution 4.2.7.1.

Consider the following truth tables:\(\ \\ \)

\begin{equation*} \begin{array}{c|c|c|c} p\amp q\amp \sim q \amp p\vee\sim q\\ \hline T \amp T\amp F \amp T\\ T \amp F\amp T \amp T\\ F \amp T\amp F \amp F\\ F \amp F\amp T \amp T\\ \end{array} \end{equation*}
\begin{equation*} \begin{array}{c|c|c|c} p\amp q\amp \sim p \amp p\vee\sim q\\ \hline T \amp T\amp F \amp T\\ T \amp F\amp F \amp F\\ F \amp T\amp T \amp T\\ F \amp F\amp T \amp T\\ \end{array} \end{equation*}

So notice \(p\vee \sim q\) is false when \(p=F, q=T\text{,}\) but \(\sim p \vee q\) is true under those circumstances. This would be enough to show they are not the same, but also note when \(p=T, q=F\text{,}\) \(p\vee \sim q\) is true but \(\sim p\vee q\) is false.
