Example 8.1.2. Element or not of a set.

Consider the set \(A=\{2,3,5\}.\)

  • \(1\not\in A.\) since 1 is not in \(A\text{.}\)
  • \(2\in A.\) since 2 is in \(A\text{.}\)
  • \(3\in A.\) since 3 is in \(A\text{.}\)
  • \(4\not\in A.\) since 4 is not in \(A\text{.}\)
  • \(5\in A.\) since 5 is in \(A\text{.}\)
