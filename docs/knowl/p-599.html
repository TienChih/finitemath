Well, If I owe \(X\) dollars in any given month, then the next month I'll owe \(X+0.05X=(1.05)X\) dollars. In another month I'll owe \(1.05X+(.05)(1.05)X=(1.05)(1.05)X=X(1.05)^2\) dollars. If we apply this logic to our situation, we can see:

\begin{equation*} \begin{array}{|c|c|} \hline \text{ Month } \amp \text{ Debt } \\ \hline 0 \amp 100\\ 1\amp 100+(.05)(100)=(1.05)100\\ 2\amp (1.05)100+(.05)((1.05)100)=(1.05)(1.05)100=(1.05)^2 100\\ 3 \amp (1.05)^2100+(.05)((1.05)^2100)=(1.05)(1.05)^2 100=(1.05)^3 100\\ \vdots \amp \vdots\\ n \amp (1.05)^n100\\ \hline \end{array} \end{equation*}
