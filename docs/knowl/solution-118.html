Solution 6.4.3.1.

The numerator has degree 3 and denominator has degree 2, so the long term behavior can be approximated by \(f(x)\approx\frac{x^3}{x^2}\approx x\) and so the long term behavior approximates \(y=x\text{.}\)

The roots of the numerator are \(0, 6\) so \(f(0)=0=f(6)\text{.}\) The root \(x=0\) has multiplicity 2, which is even, so the function does not change signs here. The root \(x=6\) has multiplicty 1, which is odd, so the function changes sign at this value.

The roots of the numerator are \(-1, 8\) so we have vertical asymptotes \(x=-1, x=8\text{.}\) These both have multiplicty 1, which is odd, so the function changes sign at these values.
