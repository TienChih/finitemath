Remark 8.2.3.

We note the following about cardinality:

  • \(|A^c|=|\mathcal{U}|-|A|\text{,}\) since \(A^c\) consists of all elements of \(\mathcal{U}\) not in \(A\text{.}\)
  • \(|\emptyset|=0\) since the empty set contains no elements.

Let us consider \(|A\cup B|.\) To see this, let's consider ExampleÂ 8.1.12 and let

\begin{equation*} \mathcal{U}=\{Ailca, Bryan, Corrine, Damien, Esmeralda, Francis\}. \end{equation*}

Then let \(A=\{Bryan, Esmeralda\}, B=\{Bryan, Francis\}\text{.}\)

We note that \(|A|=2, |B|=2\text{.}\) But if we just took \(2+2\text{,}\) we note that Bryan is counted in both sets, but the cardinality of \(\{Bryan, Esmeralda, Bryan, Francis\}=3\text{,}\) since there are only 3 elements in \(A\cup B\text{.}\) We note then that we double counted the element(s) in both sets, that is \(A\cap B\text{.}\)

We can see that there is one element in \(A\) that's not in \(B\text{,}\) one element in \(B\) that's not in \(A\text{,}\) one element in both \(A\) and \(B\text{,}\) and three elements neither in \(A\) nor \(B\text{.}\)
