We note that in our rocket situation, there is nothing particularly special about time \(x=0\text{.}\) In principle, the derivative described in Definition 7.3.1 could be applied to any value of \(x\text{.}\) Also, the rocket has a velocity at any time, and we should be able to compute it. If we apply Definition 7.3.1 to a general \(f(x)\text{,}\) we obtain:

\begin{equation*} f'(x)=\limit{h}{0}\frac{f(x+h)-f(x)}{h}\text{.} \end{equation*}
