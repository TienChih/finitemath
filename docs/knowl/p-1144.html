Let \(C\) denote “uses cream” and \(S\) denote “uses sugar”. We are given \(P(S|C)=0.8, P(S|C^c)=0.3, P(C)=0.7.\)

  1. This is \(P(S)\text{.}\) So:
    \begin{align*} P(S)\amp =P(S|C)P(C)+P(S|C^c)(1-P(C))\\ P(S)\amp =(0.8)(0.7)+(0.3)(1-0.7)=0.65 \end{align*}
    Which we can see: Moreover, we now know \(P(C)=0.7, P(S)=0.65, P(S\cap C)=P(S|C)P(C)=(0.8)(0.7)=0.56.\) So we can fill out the following venn diagram:
  2. This is \(P(C|S)=\frac{P(C\cap S)}{P(S)}=\frac{0.56}{0.65}\approx 0.861538\text{.}\)
  3. This is \(P(C^c\cap S^c)=0.21\text{.}\)
