To see this, consider the plots of \(y=x^3\) and \(y=1000x^2\text{.}\) One thousand is a huge coefficient for sure! But will that matter so much when \(x\) is in the thousands, the millions, the trillions? Eventually \(x\cdot x^2\) will be much greater than \(1000\cdot x^2\) when \(x\) is big enough. So if we plotted \(y=x^3-1000x^2\) next to \(y=x^3\) We see that they have the same general shape.
