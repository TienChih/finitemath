Solution 9.5.3.1.

We note \(n=10, p=0.1\text{,}\) let \(X\) denote the number of malfunctioning machines.

  1. This is \(P(X=2)=C(10,2)(0.1)^2(0.9)^8=0.1937\)
  2. At least 1 means 1 though 10. So one could sum up \(P(X=1)+P(X=2)+\cdots P(X=10)\text{.}\) But this is also every possibility but \(P(X=0)\text{.}\) So we could do
    \begin{equation*} 1-P(X=0)=1-C(10,0)(0.1)^0(0.9)^{10}=1-0.3487=0.6513. \end{equation*}
