Item 4.1.1.2.
A truth table could be constructed as follows:\(\ \\ \)
\begin{equation*} \begin{array}{c|c|c|c} a\amp b\amp \sim b \amp a\wedge \sim b\\ \hline T \amp T\amp F \amp F\\ T \amp F\amp T \amp T\\ F \amp T\amp F \amp F\\ F \amp F\amp T \amp F\\ \end{array} \end{equation*}
Which is only true if \(a\) is true (he eats nuggets) and \(b\) is false (he does not potty).
