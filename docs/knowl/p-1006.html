First note that \(V(t)=60000(1+0.5t+0.1t^2)^{-1}\text{.}\) So the truck's value changes according to:

\begin{align*} V'(t)\amp = \frac{d}{dt}[60000(1+0.5t+0.1t^2)^{-1}]\\ \amp = 60000(-1)(1+0.5t+0.1t^2)^{-2}\frac{d}{dt}[1+0.5t+0.1t^2]\\ \amp = 60000(-1)(1+0.5t+0.1t^2)^{-2}(0.5+0.2t)\\ \amp = \frac{-30000-12000t}{(1+0.5t+0.1t^2)^{2}}\text{.} \end{align*}

Thus:

  1. \(N'(3)=\frac{-30000-12000(3)}{(1+0.5(3)+0.1(3)^2)^{2}}\approx -5709.34\text{.}\) So the truck is losing \(\$5709.34\) a year.
  2. \(N'(10)=\frac{-30000-12000(10)}{(1+0.5(10)+0.1(10)^2)^{2}}\approx -585.94\text{.}\) So the truck is losing \(\$585.94\) a year.
