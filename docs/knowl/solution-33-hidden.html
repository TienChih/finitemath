We note \(U(t)=mt+b\text{.}\) Moreover, initially when \(t=0\) there were 2,000 users so \(U(0)=2000\) and \(b=2000\text{.}\) To find \(m\text{:}\)

\begin{equation*} m=\frac{\Delta U}{\Delta t}=\frac{5000-2000}{5-0}=\frac{3000}{6}=500\text{.} \end{equation*}

So the app gains 600 users per week and: \(U(t)=600t+2000\)
