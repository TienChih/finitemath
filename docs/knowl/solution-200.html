Solution 9.3.3.1.

Consider:

  1. If \(D\) is the event “at least one diamond”, then \(D^c\) means “no diamonds”. \(P(D^c)=\frac{C(39,5)}{C(52,5)}=\frac{575757}{2598960}\approx 0.2215336\text{,}\) since there are 29 non diamonds to pick 5 from. So \(P(D)=1-P(D^c)=1-\frac{C(39,5)}{C(52,5)}\approx 0.7784664.\)
  2. Let \(B\) denote “all black cards”, and \(N\) denote “all non-face cards”. Note \(P(B)=\frac{C(26,5)}{C(52,5)}\text{,}\) \(P(N)=\frac{C(36,5)}{C(52,5)}\text{,}\) and \(P(B\cap N)=\frac{C(18,5)}{C(52,5)}\text{.}\) Thus
    \begin{equation*} P(B\cup N)=P(B)+P(N)-P(B\cap N)=\frac{C(26,5)+C(36,5)-C(18,5)}{C(52,5)}=\frac{65780+376992-8568}{2598960}\approx 0.167068. \end{equation*}
  3. This is \(P(N|B)=\frac{P(N\cap B)}{P(B)}=\frac{C(18,5)/C(52,5)}{C(26,5)/C(52,5)}=\frac{C(18,5)}{C(26,5)}=\frac{8568}{65780}\approx 0.130252.\)
