Solution 10.3.4.6.

We are given a probability and asked to find a bound. Specifically, the bounds of a region that is symmetric about the mean, the same distance to the left as to the right. So: \(P(-z_0\lt Z\lt z_0)=0.8\) is satisfied when \(z_0\approx 1.2816\)

Note that since 80% of the area falls in this central region, the sum of the tails must be 20%, and since the tails are mirror images of each other, each tail must contain 10%. Thus \(z_0\) must satisfy \(P(Z\lt -z_0)=0.10\text{:}\) Which yields \(-z_0\approx -1.2816\) so the same \(z_0\approx 1.2816\text{.}\) We could also approach this from the right:
