Example 9.4.2.

Consider the experiment rolling two dice. We can see that the sample space for this experiment is:

\begin{equation*} (1,1),(1,2),(1,3),(1,4),(1,5),(1,6) \end{equation*}
\begin{equation*} (2,1),(2,2),(2,3),(2,4),(2,5),(2,6) \end{equation*}
\begin{equation*} (3,1),(3,2),(3,3),(3,4),(3,5),(3,6) \end{equation*}
\begin{equation*} (4,1),(4,2),(4,3),(4,4),(4,5),(4,6) \end{equation*}
\begin{equation*} (5,1),(5,2),(5,3),(5,4),(5,5),(5,6) \end{equation*}
\begin{equation*} (6,1),(6,2),(6,3),(6,4),(6,5),(6,6) \end{equation*}

So if we consider the random variable \(X\) to be “sum of die outcomes”, we would have the assignment:

\begin{equation*} 2,3,4,5,6,7 \end{equation*}
\begin{equation*} 3,4,5,6,7,8 \end{equation*}
\begin{equation*} 4,5,6,7,8,9 \end{equation*}
\begin{equation*} 5,6,7,8,9,10 \end{equation*}
\begin{equation*} 6,7,8,9,10,11 \end{equation*}
\begin{equation*} 7,8,9,10,11,12 \end{equation*}

On the other hand, if the random variable \(Y\) to be “second die outcome”, we would have the assignment:

\begin{equation*} 1,2,3,4,5,6 \end{equation*}
\begin{equation*} 1,2,3,4,5,6 \end{equation*}
\begin{equation*} 1,2,3,4,5,6 \end{equation*}
\begin{equation*} 1,2,3,4,5,6 \end{equation*}
\begin{equation*} 1,2,3,4,5,6 \end{equation*}
\begin{equation*} 1,2,3,4,5,6 \end{equation*}

There are many more possibilities of course, but one last one, if \(Z\) denoted “larger of die rolls” we would have:

\begin{equation*} 1,2,3,4,5,6 \end{equation*}
\begin{equation*} 2,2,3,4,5,6 \end{equation*}
\begin{equation*} 3,3,3,4,5,6 \end{equation*}
\begin{equation*} 4,4,4,4,5,6 \end{equation*}
\begin{equation*} 5,5,5,5,5,6 \end{equation*}
\begin{equation*} 6,6,6,6,6,6 \end{equation*}
