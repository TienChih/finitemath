We can see this here with the following Desmos graph. Initially, \(P(A)=0.6, P(B)=0.4, P(A\cap B)=0.3, P(A\cup B)=0.7\text{.}\) We see that while \(P(A)\) takes up 60% of the \([0,1]\) interval, it takes up 75% of the \(P(B)\) interval, so \(P(A|B)=\frac{0.3}{0.4}=0.75.\) Similarly \(P(B)\) takes up 40% of the \([0,1]\) interval, it takes up 50% of the \(P(A)\) interval, so \(P(B|A)=\frac{0.3}{0.6}=0.5.\) Adjust the probablities, and slide the regions to change the overlap to see how that effects the values!
