Why would we care about a second formulation, and how do we know it's even a line? To answer the first question, we recall that ANY geometric object defined by an equation is the set of points that make the equation true. So given the equation \(y-y_0=m(x-x_0)\text{,}\) if we plug in \((x_0, y_0)\text{,}\) we would get \(y_0-y_0=m(x_0-x_0)\) or \(0=0\) which is definitely a true statement. So whatever shape we get from \(y-y_0=m(x-x_0)\text{,}\) it contains \((x_0, y_0)\text{.}\)
