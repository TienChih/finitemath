Using the point-slope form, knowing that \(\color{blue}{(2,5)}\) lies on the line, and that \(\color{red}{m}\text{:}\)

\begin{align*} y-y_0\amp = m(x-x_0)\\ y-\color{blue}{5}\amp = \color{red}{3}(x-\color{blue}{2})\\ y\amp = 3x-6+5\\ y\amp = 3x-1\text{.} \end{align*}

Either way, you obtain the same line, with slope 3, passing through (2,5).
