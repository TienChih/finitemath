There are several approaches. You only need to go through one of these to obtain answer.

First note that \(P(B)=\frac{8}{25}=0.32\) and \(P(B|A)=\frac{5}{12}\approx 0.41666667\text{.}\) These are not the same (being a biology major seems to improve the odds that you are an athlete) so these are not independent.

Next note that \(P(A)=\frac{12}{25}=0.48\) and \(P(A|B)=\frac{5}{8}=0.625\text{.}\) These are not the same (being a student athelete seems to improve the odds that you are a biology major) so these are not independent.

Finally note that \(P(A)=\frac{12}{25}=0.48, P(B)=\frac{8}{25}=0.32, P(A\cap B)=\frac{5}{25}=0.2\) but \(P(A)P(B)=0.48\cdot 0.32=0.1536\neq 0.2=P(A\cap B)\text{.}\) So these are not independent.
