Suppose we were trying to maximize the function \(f(x,y)=x+y \) subject to the following constraints:

\begin{align*} 3x+2y\amp \leq 8\\ x-y\amp \leq 3\\ x\amp \geq 0\\ y\amp \geq 0 \end{align*}

We can see from the Desmos graph below the region of points that satisfy the above system. This is called the feasible region, or the collection of points which could be a solution. We can also drag \((a,b)\) around to see what the value of \(f(x,y)\) would be for any point.
