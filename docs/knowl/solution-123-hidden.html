There are several approaches. You only need to go through one of these to obtain answer. Also recall that 4 people ordered both a burger and a hotdog.

First note that \(P(B)=\frac{18}{20}=0.9\) and \(P(B|H)=\frac{4}{6}\approx 0.66666667\text{.}\) These are not the same (ordering a hotdog makes it less likely you ordered a buger) so these are not independent.

Next note that \(P(H)=\frac{6}{20}=0.3\) and \(P(H|B)=\frac{4}{18}=0.22222\text{.}\) These are not the same (ordering a hamburger makes it less likelt you also got a hotdog) so these are not independent.

Finally note that \(P(B)=\frac{18}{20}=0.9, P(H)=\frac{6}{20}=0.3, P(B\cap H)=\frac{4}{20}=0.2\) but \(P(B)P(H)=0.9\cdot 0.3=0.27\neq 0.2=P(B\cap H)\text{.}\) So these are not independent.
