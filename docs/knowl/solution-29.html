Solution 1.4.4.3.

The equilibrium point is the point where the quantity supplied and demanded are the same. Algebraically, this means \(D(p)=S(p)\text{,}\) and so:

\begin{align*} -5p+5200\amp = 3p\\ 5200\amp = 8p\\ p\amp = \frac{5200}{8}=650\text{.} \end{align*}

So the equilibrium happens when \(p=650\) that is $650 per unit. Then, we note that \(S(650)=3*650=1950\) and \(D(p)=-5*650+5200=1950\text{,}\) so the equilibrium point is \((650, 1950)\) or \(\dollar650\) per unit, and 1950 units sold.
