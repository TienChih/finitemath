Sliding \(k\) shifts the parabola up and down. We can think of this as adding to or detracting from the value of \(x^2\text{.}\) Sliding \(h\) shifts the graph left and right. We can think of this as slowing down or speeding up \(x^2\) by \(h\) units. \(a\) stretches or squashes \(x^2\text{,}\) or if \(a\) is negative, also reflects this across the \(x\) axis. But all of these transformations preserve the fundamental shape of a parabola, so all quadratic functions have this shape.
