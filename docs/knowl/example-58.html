Example 6.1.9. Domain and Range of a function.

Consider the domain and range of \(h(x)=\sqrt{2+x} \text{.}\) In this case, we note that we may take the squareroot of zero, but not negative numbers. We note then that \(2+x\geq0\) for \(h(x)\) to be defined. Thus the domain is \((-2,\infty)\text{,}\) every real number greater than or equal to -2 is a valid input.

Similarly, the output of the square root function is non-negative, and thus the range is \([0,\infty)\text{,}\) every real value greater than or equal to zero is a valid output.
