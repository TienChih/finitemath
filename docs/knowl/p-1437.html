Will this always happen? Consider the transition matrix

\begin{equation*} M=\begin{pmatrix} 0\amp 1 \ 1 \amp 0\end{pmatrix}\text{.} \end{equation*}

It has stable solution when

\begin{equation*} \begin{pmatrix} -1\amp 1 \ 1 \amp 0-1\end{pmatrix}\begin{pmatrix} x \\ y\end{pmatrix}=\begin{pmatrix}0\\0\end{pmatrix} \end{equation*}

which occurs when \(x=y\text{.}\) So if our initial state is \(x=5, y=5\text{,}\) in 100 iterations:

We see that we have stability. But what about if our initial state were \(x=10, y=0\text{?}\)
We see that we oscilate between states \(\begin{pmatrix} 10\\0\end{pmatrix}\) and \(\begin{pmatrix} 0\\10\end{pmatrix}\text{.}\) We do not achieve equilibrium.
