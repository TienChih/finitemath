Let \(G\) denote “grad school” and \(W\) denote “working”. We are given \(P(W|G)=0.6, P(W|G^c)=0.9, P(G)=0.15.\) We are asked to find \(P(G \cup W)\text{.}\) We first find \(P(W)\text{:}\)

\begin{align*} P(W)\amp =P(W|G)P(G)+P(W|G^c)(1-P(G))\\ \amp =(0.6)(0.15)+(0.9)(1-0.15)=0.855 \end{align*}

Which we can see: Moreover, we now know \(P(G)=0.15, P(W)=0.855, P(G\cap W)=P(W|G)P(G)=(0.6)(0.15)=0.09\) So we can fill out the following venn diagram:

So \(P(W\cup G)=0.06+0.765+0.09=0.915\) or \(P(W\cup G)=P(W)+P(G)-P(W\cap G)=0.855+0.15-0.09=0.915\text{.}\)
