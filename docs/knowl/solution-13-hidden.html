We start by finding the slope. As before:

\begin{equation*} m=\frac{\Delta y}{\Delta x} =\frac{4-(-2)}{6-1}=\frac{6}{5}=1.2. \end{equation*}

We then have options. Knowing that \(m=1.2\) and that we pass through \((1,-2),(6,4)\text{,}\) we can solve for a \(y-\)intercept:

\begin{align*} g(x)\amp = mx+b\\ -2\amp = 1.2(1)+b\\ b\amp = -2-1.3=-3.2\\ g(x)\amp = 1.2x-3.2. \end{align*}

Or, more directly:

\begin{align*} g(x)-y_0\amp = m(x-x_0)\\ g(x)-(-2)\amp = 1.2(x-1)\\ g(x)+2\amp = 1.2x-1.2\\ g(x)\amp = 1.2x-3.2. \end{align*}
