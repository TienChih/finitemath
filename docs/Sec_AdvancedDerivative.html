Section 7.5 Advanced Derivatve Techniques

Subsection 7.5.1 Product Rule

Since the behavior of derivatives of sums behave so well, its tempting to believe that the derivatives of products would behave similarly well, that is, it may be reasonable to guess that \(\frac{d}{dx}[f(x)g(x)]=f'(x)g'(x)\text{.}\) To try to see if this is true, we should ask, is:

\begin{equation*} \frac{d}{dx}[x\cdot x]=\frac{d}{dx}[x]\frac{d}{dx}[x]? \end{equation*}

Well on the left hand side \(\frac{d}{dx}[x\cdot x]=\frac{d}{dx}[x^2]=2x\text{.}\) On the right, \(\frac{d}{dx}[x]\frac{d}{dx}[x]=1\cdot1=1\text{,}\) so these are clearly not the same. It's safe to say that in general

\begin{equation*} \frac{d}{dx}[f(x)g(x)]\neq f'(x)g'(x). \end{equation*}
Remark 7.5.1.

We do have a rule for the derivative of products. The product rule and it states \(\frac{d}{dx}[f(x)g(x)]=f'(x)g(x)+f(x)g'(x)\text{.}\) To see why:

\begin{align*} \frac{d}{dx}[f(x)g(x)]\amp = \limit{h}{0}\frac{f(x+h)g(x+h)-f(x)g(x)}{h},\ \text{ at this stage, I'm going to pull a clever trick and add by 0: }\\ \amp = \limit{h}{0}\frac{f(x+h)g(x+h)\color{blue}{-f(x)g(x+h)+f(x)g(x+h)}-f(x)g(x)}{h}\\ \amp = \limit{h}{0}\frac{f(x+h)g(x+h)-f(x)g(x+h)}{h}+\limit{h}{0}\frac{f(x)g(x+h)-f(x)g(x)}{h}\\ \amp = \limit{h}{0}\frac{f(x+h)-f(x)}{h}g(x+h)+\limit{h}{0}f(x)\frac{g(x+h)-g(x)}{h}\\ \amp = f'(x)g(x)+f(x)g'(x)\text{.} \end{align*}
Example 7.5.2.

Compute the derivative of \(f(x)=x^3(x+1)\text{.}\)

Solution

Utilizing Remark 7.5.1:

\begin{align*} f'(x)\amp = \frac{d}{dx}[x^3](x+1)+x^3\frac{d}{dx}[x+1]\\ \amp = 3x^2(x+1)+x^3(1)\\ \amp = 3x^3+3x^2+x^3\\ \amp = 4x^3+3x^2\text{.} \end{align*}

Also note we could have computed this some other way:

\begin{align*} f(x)\amp = x^3(x+1)=x^4+x^3\\ f'(x)\amp = 4x^3+3x^2\text{.} \end{align*}

Subsection 7.5.2 Quotient Rule

It should seem clear that

\begin{equation*} \frac{d}{dx}[\frac{f(x)}{g(x)}]\neq \frac{f'(x)}{g'(x)}. \end{equation*}
Remark 7.5.3.

We have an analagous rule for the derivative of quotients. The quotient rule and it states \(\frac{d}{dx}[\frac{f(x)}{g(x)}]=\frac{f'(x)g(x)-f(x)g'(x)}{g(x)^2}\text{.}\) To see why, let \(Q(x)=\frac{f(x)}{g(x)}\) and observe:

\begin{align*} Q(x)\amp = \frac{f(x)}{g(x)}\\ Q(x)g(x)\amp = f(x),\ \text{ now we can differentiate both sides }\\ \frac{d}{dx}[Q(x)g(x)]\amp = f'(x),\ \text{ then by the product rule: }\\ Q'(x)g(x)+Q(x)g'(x)\amp = f'(x)\\ Q'(x)g(x)\amp = f'(x)-Q(x)g'(x),\ \text{then recall that \(Q(x)=\frac{f(x)}{g(x)}\)}\\ Q'(x)g(x)\amp = f'(x)-\frac{f(x)g'(x)}{g(x)}\\ Q'(x)g(x)\amp = \frac{f'(x)g(x)-f(x)g'(x)}{g(x)}\\ Q'(x)\amp = \frac{f'(x)g(x)-f(x)g'(x)}{g(x)^2}\text{.} \end{align*}
Example 7.5.4.

Compute the derivative of \(g(x)=\frac{x}{x^2+1}\text{.}\)

Solution

Utilizing Remark 7.5.3:

\begin{align*} g'(x)\amp = \frac{(x^2+1)\frac{d}{dx}[x]-x\frac{d}{dx}[x^2+1]}{(x^2+1)^2}\\ \amp = \frac{(x^2+1)(1)-x(2x)}{(x^2+1)^2}\\ \amp = \frac{1-x^2}{(x^2+1)^2}\text{.} \end{align*}

Subsection 7.5.3 Chain Rule

We finally have a rule for composition of functions:

Remark 7.5.5.

The chain rule states

\begin{equation*} \frac{d}{dx}[f(g(x))]=f'(g(x))g'(x). \end{equation*}

To see why:

\begin{align*} \frac{d}{dx}[f(g(x))]\amp = \limit{h}{0}\frac{f(g(x+h))-f(g(x))}{h},\ \text{ Similar to the product rule, we will multiply by 1: }\\ \amp = \limit{h}{0}\frac{f(g(x+h))-f(g(x))}{h}\color{blue}{\frac{g(x+h)-g(x)}{g(x+h)-g(x)}}\\ \amp = \limit{h}{0}\frac{f(g(x+h))-f(g(x))}{g(x+h)-g(x)}\frac{g(x+h)-g(x)}{h}\\ \amp = \limit{h}{0}\frac{f(g(x+h))-f(g(x))}{g(x+h)-g(x)}\limit{h}{0}\frac{g(x+h)-g(x)}{h}\text{.} \end{align*}

We're going to play a second trick here, we are going to let \(k=g(x+h)-g(x)\text{,}\) we also note then that \(g(x+h+=g(x)+k\text{.}\) Also note that when \(h\to 0\) It follows that \(k=g(x+h)-g(x)\to g(x+0)-g(x)=0\text{.}\) So we can make some replacements:

\begin{align*} \amp = \limit{h}{0}\frac{f(g(x+h))-f(g(x))}{g(x+h)-g(x)}\limit{h}{0}\frac{g(x+h)-g(x)}{h}\\ \amp = \limit{\color{red}{k}}{0}\frac{f(\color{red}{g(x)+k})-f(g(x))}{\color{red}{k}}\limit{h}{0}\frac{g(x+h)-g(x)}{h}\\ \amp = f'(g(x))g'(x) \end{align*}
Example 7.5.6.

Compute the derivative of \(f(x)=\sqrt{x^2+1}\text{.}\)

Solution

We can think of this as \(f(x)=g(h(x))\) where \(g(x)=\sqrt{x}=x^{1/2}, h(x)=x^2+1\text{.}\) So \(g'(x)=\frac{1}{2}x^{-1/2}, h'(x)=2x\text{.}\) Utilizing Remark 7.5.5:

\begin{align*} f'(x)\amp = \frac{d}{dx}[\sqrt{x^2+1}]\\ \amp = \frac{d}{dx}[g(h(x))]\\ \amp = g'(h(x))h'(x)\\ \amp = \frac{1}{2}(x^2+1)^{-1/2}(2x)\\ \amp = \frac{x}{\sqrt{x^2+1}}\text{.} \end{align*}

Subsection 7.5.4 Exercises

Checkpoint 7.5.7.

The total profit \from selling \(x\) widgets is:

\begin{equation*} P(x)=\$ \frac{200x-1000}{20x+30}. \end{equation*}

Find the marginal average cost function.

Solution

If you sell \(x\) widgets, the total profit is \(P(x)=\$ \frac{200x-1000}{20x+30}.\) So the average profit per widget is

\begin{equation*} A(x)=\frac{P(x)}{x}= \frac{200x-1000}{20x+30}\cdot\frac{1}{x}=\frac{200x-1000}{20x^2+30x}. \end{equation*}

Then, the marginal average cost is computed via the derivative:

\begin{align*} A'(x)\amp = \frac{d}{dx}[\frac{200x-1000}{20x^2+30x}]\\ \amp = \frac{\frac{d}{dx}[200x-1000](20x^2+30x)-(200x-1000)\frac{d}{dx}[20x^2-30x]}{(20x^2+30x)^2}\\ \amp = \frac{200(20x^2+30x)-(200x-1000)(40x-30)}{(20x^2+30x)^2}\\ \amp = \frac{4000x^2+6000x-(8000x^2-34000x-30000)}{(20x^2+30x)^2}\\ \amp = \frac{4000x^2+6000x-8000x^2+34000x+30000}{(20x^2+30x)^2}\\ \amp = \frac{-4000x^2+40000x+30000}{(20x^2+30x)^2}\text{.} \end{align*}

Checkpoint 7.5.8.

Assume that the total number (in millions) of bacteria present in a culture at a certain time \(t\) (in hours) is given by

\begin{equation*} N(t)=3t(t-8)^2+30. \end{equation*}
  1. Find \(N'(t)\text{.}\)
  2. How fast is the population of bacteria growing in 10 hours?
Solution

Consider:

  1. We compute \(N'(t)\text{:}\)
    \begin{align*} N'(t)\amp = \frac{d}{dt}[3t(t-8)^2+30]\\ \amp = \frac{d}{dt}[3t(t-8)^2]+\frac{d}{dt}[30]\\ \amp = \frac{d}{dt}[3t](t-8)^2+3t\frac{d}{dt}[(t-8)^2]+0\\ \amp = 3(t-8)^2+3t\frac{d}{dt}[t^2-16t+64]\\ \amp = 3(t^2-16t+64)+3t(2t-16))\\ \amp = 3t^2-48t+192+6t^2-48t\\ \amp = 9t^2-96t+192\text{.} \end{align*}
  2. Notice that \(N(10)=9(10^2)-96(10)+192=132\) or 132 million bacteria per hour.

Checkpoint 7.5.9.

A company purchases a truck depreciates according to the formula

\begin{equation*} V(t)=\frac{60000}{1+0.5t+0.1t^2} \end{equation*}

where \(V\) is measured in dollars and \(t\) is years after purchase.

Find the rate the truck's value is changing in:

  1. Three years.
  2. Ten years.
Solution

First note that \(V(t)=60000(1+0.5t+0.1t^2)^{-1}\text{.}\) So the truck's value changes according to:

\begin{align*} V'(t)\amp = \frac{d}{dt}[60000(1+0.5t+0.1t^2)^{-1}]\\ \amp = 60000(-1)(1+0.5t+0.1t^2)^{-2}\frac{d}{dt}[1+0.5t+0.1t^2]\\ \amp = 60000(-1)(1+0.5t+0.1t^2)^{-2}(0.5+0.2t)\\ \amp = \frac{-30000-12000t}{(1+0.5t+0.1t^2)^{2}}\text{.} \end{align*}

Thus:

  1. \(N'(3)=\frac{-30000-12000(3)}{(1+0.5(3)+0.1(3)^2)^{2}}\approx -5709.34\text{.}\) So the truck is losing \(\$5709.34\) a year.
  2. \(N'(10)=\frac{-30000-12000(10)}{(1+0.5(10)+0.1(10)^2)^{2}}\approx -585.94\text{.}\) So the truck is losing \(\$585.94\) a year.
