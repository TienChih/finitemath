Section 6.6 Logarithms

So looking at any exponential function, but specifically for increasing ones, we notice that they are 1-1, meaning two different inputs give you 2 different outputs. Thus, \(f(x)=a^x\) is an invertible function. You can see in these graphs that reflecting the exponential over the \(y=x\) line giving us another function: We call this function \(\log_a(x)\text{.}\)

Subsection 6.6.1 Properties of Logarithms

As the inverse of an exponential function, it has some properties:

  • It's domain is only positive numbers. Why? Because the only possible outputs of exponential outputs are positive numbers. Taking the inverse reverses the roles of the domain and range, and so the domain of \(\log_a\) is the positives.

  • As \(\mathbf{x}\) goes to 0, \(\mathbf{\log_a(x)}\) goes to \(\mathbf{-\infty}\text{.}\) This is the result of the reflection. Normally \(a^x\) (for \(a>1\)) goes to 0 as \(x\to -\infty\text{,}\) so when reflected, that line is now asymptotic to the \(y\)-axis. Also \(\log_a\) returns the power necessary to achieve the value \(x\text{.}\) If \(x\) is a small number like 0.0001, what power would I have to raise \(a\) to to get \(0.0001\text{?}\) It can't be 0, \(a^0=1\text{,}\) so it has to be less than 0, and the smaller \(x\) is, the lower this power must go.

  • As \(\mathbf{x\to\infty, \log_a(x)\to \infty}\text{,}\) again this is the result of the reflection, but we can also think of this as the powers \(a\) needs to be raised to in order to achieve \(x\text{,}\) as this increases, those powers must increase as well.

  • \(\log_a(xy)=\log_a(x)+\log_a(y)\text{.}\) Too see this consider:

    \begin{align*} xy\amp = a^{\log_a{xy}}\\ xy\amp = x\cdot y\\ \amp = a^{\log_a(x)}\cdot a^{\log_a(y)}\\ \amp = a^{\log_a(x)+\log_a(y)} \end{align*}

  • \(\log_a(x^y)=y\log_a(x)\text{.}\) To see this, consider:

    \begin{align*} x^y\amp = a^{\log_a(x^y)}.\\ x^y\amp = (a^{\log_a(x)})^y\\ \amp = a^{y\log_a(x)} \end{align*}

Typically most textbooks include a whole bunch of other arithmetic rules, but they can all be distilled from the 2 above so they're all totally pointless.

Subsection 6.6.2 Utility of logs

Since each logarithm is an inverse of an exponential function, and exponential equation has an equivalent logathrithmic equation.

\begin{equation*} \begin{array}{|c|c|} \hline \text{Exponential Equation} \amp \text{Logarithmic Equation}\\ \hline 2^3=8\amp \log_2(8)=3\\ 10^2=100\amp \log_{10}(100)=2\\ (0.9)^2=0.81\amp \log_{0.9}(0.81)=2\\ 9^{1/2}=3\amp \log_3(9)=\frac{1}{2}\\ 2^{-2}\frac{1}{4}\amp \log_2(\frac{1}{4})=-2\\ 3^x=4\amp \log_3(4)=x\\ a^b=c\amp \log_a(c)=b\\ \hline \end{array} \end{equation*}

Special cases of logs are log base 10 which is usually just denoted \(\log\) and log based \(e\text{,}\) which is denoted \(\ln\text{.}\) Astronomers and other scientists like \(\log\) since taking \(\log\) base 10 gives you approximate magnitude of stuff. As a mathematician, I prefer \(\ln\) since \(e\) has pretty special mathematical properties, plus it's shorter, which makes it better.

The main use of logs algebraically is to de-exponentiate expressions. So if one is trying to solve \(10=2^x\text{,}\) we could do this via:

\begin{align*} 10\amp = 2^x\\ \ln(10)\amp = \ln(2^x)\\ \ln(10)\amp = x\ln(2)\\ x\amp = \frac{\ln(10)}{\ln(2)}\approx 3.3219\text{.} \end{align*}

Subsection 6.6.3 Computations

Checkpoint 6.6.1.

Solve for \(x\text{:}\)

  1. \(e^{x+2}=100\text{.}\)
  2. \(\displaystyle \log(x)+\log(x+3)=1\)
Solution

Consider the following:

  1. Starting with:
    \begin{align*} e^{x+2}\amp = 100\\ \ln(e^{x+2})\amp = \ln(100)\\ x+2\amp = \ln(100)\\ x\amp = \ln(100)-2\approx 2.60517 \end{align*}
  2. Recall that \(\log\) is \(\log_{10}\text{.}\)
    \begin{align*} \log(x)+\log(x+3)\amp = 1\\ \log(x(x+3))\amp = 1\\ 10^{\log(x(x+3))}\amp = 10^1\\ x(x+3)\amp = 10\\ x^2+3x-10\amp = 0\\ (x+5)(x-2)\amp = 0\\ x\amp = -5, 2 \end{align*}
Checkpoint 6.6.2.

Suppose a resturaunt's weekly revenue in thousands of dollars is \(R(t)=50-30e^{-0.1t}\) in \(t\) weeks after opening.

  1. How much revenue did it make in it's initial week?
  2. When will the resturaunt generate \(\$40\) thousand a week?
Solution

Consider the following:

  1. In week 0, we have \(R(0)=50-30e^0=20\) or \(\$20\) thousand dollars.

  2. We solve:

    \begin{align*} R(t)=50-30e^{-0.1t}\amp = 40\\ R(t)10\amp =30e^{-0.1t}\\ e^{-0.1t}\amp = \frac{1}{3}\\ \ln(e^{-0.1t})\amp = \ln(\frac{1}{2})\\ -0.1t\amp = \ln(\frac{1}{3})\\ t\amp = -10\ln(\frac{1}{3})\approx 10.9861 \end{align*}

    So at almost the 7th week.

Checkpoint 6.6.3.

Consider a radioactive compound Imaginarium which has halflife of 100 minutes. Let \(I(t)\) be the amount of Imaginarium left in grams after \(t\) minutes if you started with 1000g.

  1. Find \(I(t)\text{.}\)
  2. When will you have 10g left?
Solution

Consider the following:

  1. We note that since the halflife is 100 minutes:

    \begin{align*} \frac{1}{2}\amp = a^5\\ a\amp = (\frac{1}{2})^{1/5}\approx0.87055 \end{align*}

    So \(I(t)=1000\cdot ((\frac{1}{2})^{1/5})^t=1000\cdot(\frac{1}{2})^{t/5}\text{.}\)

  2. We solve:

    \begin{align*} 10=I(t)\amp = 1000\cdot\left(\frac{1}{2}\right)^{t/5}\\ \frac{1}{100}\amp = \left(\frac{1}{2}\right)^{t/5}\\ \ln\left(\frac{1}{100}\right)\amp = \ln\left(\left(\frac{1}{2}\right)^{t/5}\right)\\ \ln\left(\frac{1}{100}\right)\amp = \frac{t}{5}\cdot\ln\left(\frac{1}{2}\right)\\ t\amp = \frac{5\ln\left(\frac{1}{100}\right)}{\ln\left(\frac{1}{2}\right)}\approx 33.21928 \end{align*}

    So in just over 33 minutes.
