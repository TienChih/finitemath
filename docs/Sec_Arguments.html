Section 4.5 Arguments

In this section, we discuss logical arguments. The point of logic, is to take certain facts or statements known to be true, and from that draw certain conclusions. We will use what we've learned about logic to describe some valid argument types. We will then put these argument types together to deduce what is and isn't a valid or appropriate argument.

Subsection 4.5.1 Argument types

The first type of argument is Modus Ponens, it is based on the implication \(p\to q\text{.}\) If you know that whenever \(p\) is true, then we can deduce \(q\) is true.

\begin{equation*} \begin{array}{l} p\to q \\ p\\ \hline q \end{array} \end{equation*}

So say “If your pet is a pitbull, then it is a dog”. “Your pet is a pitbull”. It would then be fair to conclude “Therefore your pet is a dog”.

Example 4.5.1.

The following are also examples of Modus Ponones:

  1. "If we drive faster, we will make it in time. We drove faster. Thus we made it in time."
  2. "If I get sick, I will stay home. I got sick. Therefore I stayed home."
  3. "If \(x\) is even, then \(x^2\) is even. \(x\) is even, thus \(x^2\) is even."
  4. "If \(S\) is a square, then \(S\) is a rectangle. \(S\) is a square, thus \(S\) is a rectangle."

The next argument type is Modus Tollens. This is analagous to the contrapositive. So if you're given that \(p\to q\text{,}\) and you know that \(q\) isn't true, that is \(\sim q\text{,}\) then it follows that \(p\) isn't true, or \(\sim p\text{.}\)

\begin{equation*} \begin{array}{l} p\to q \\ \sim q\\ \hline \sim p \end{array} \end{equation*}

So suppose the following were true: “If your eat peanuts, then you will have an allergic reaction” and “You do not have an allergic reaction”. It would then be fair to conclude “Therefore you did not eat peanuts”.

Example 4.5.2.

The following are also examples of Modus Tollens:

  1. "If I eat lunch, I won't be hungry. I'm hungry. Thus I must not have eaten lunch."
  2. "If we stay behind the line, no one will get hurt. Someone got hurt. We must not have stayed behind the line."
  3. "If \(x\) is even, then \(x^2\) is even. \(x^2\) is odd, thus \(x\) is odd."
  4. "If \(S\) is a square, then \(S\) is a rectangle. \(S\) is a not a rectangle, thus \(S\) must not be a square."

The next arugment type is Disjunctive Syllogism. This is based on or. If you're given that either \(p\) or \(q\) is true, that is \(p\vee q \text{,}\) and then one of them is false (\(\sim p\) or \(\sim q\)) , then it follows that the other must be true (\(q\) or \(p\)).

\begin{equation*} \begin{array}{l} p\vee q \\ \sim q\\ \hline p \end{array} \end{equation*}
\begin{equation*} \begin{array}{l} p\vee q \\ \sim p\\ \hline q \end{array} \end{equation*}

"I have to take either math or science next semester, I'm not going to take math, so therefore, I must take science."

Example 4.5.3.

The following are also examples of Disjunctive Syllogism:

  1. "Either I or my spouse has the card. I don't have it so my spouse must have it."
  2. "She can order the fish or the chicken. She didn't order the chicken so she must have ordered the fish."
  3. "A linear function \(\ell(x)\) either has nonero slope or is constant. It's not constant so \(\ell(x)\) must have nonzero slope."
  4. "An integer greater than 1 \(n\) is either prime or composite. Since \(n\) isn't prime, it must be composite."

The final type of argument is Reasoning by Transitivity. It connects two implications together. If you have \(p\to q\) and \(q\to r\text{,}\) then it must follow that \(p\to r\text{.}\)

\begin{equation*} \begin{array}{l} p\to q \\ q\to r\\ \hline p\to r \end{array} \end{equation*}

"If it rains then I will get wet. If I get wet, I will get sick." \(\to\) “Therefore, if it rains, then I will get sick.”

Example 4.5.4.

The following are also examples of Reasoning by Transitivity:

  1. "If we drive faster, we will make it in time. If we make it in time, we can have lunch. Thus if we drive faster, we can have lunch."
  2. "If I get sick, I will stay home. If I stay home, I will work on my book. So if I get sick, I will work on my book."
  3. "If it rains, I will have to move the clothes inside. If I move the clothes inside, I will be late. So if it rains, I will be late."
  4. "If \(S\) is a square, then \(S\) is a rectangle. If \(S\) is a rectangle, all it's angles are right angles. Thus if \(S\) is a square, then all of it's angles are right angles."
Checkpoint 4.5.5.

Identify the argument type for each of the following:

  1. "I can either pay the gas or electric bil. I didn't pay gas so I must have paid the electric bill."
  2. "If it snows, the children can go sledding, they didn't go sledding, thus it didn't snow."
  3. "If it snows, the children can go sledding, if the children go sledding the house will be quiet. Thus if it snows then the house will be quiet."
  4. "If the firm makes enough money, then we will hire new staff members. The firm enough money, thus we will hire new staff members."
  5. "If \(x\) is a multiple of 4, then \(x\) is not prime. \(x\) is prime, thus \(x\) is not a multiple of four."
  6. "If a function has a constant slope, then it is linear. If a function is linear, then it is a polynomial. Thus every function with positive slope is a polynomial."
  7. "All nonzero numbers \(x\)are either positive or negative. If \(x\) is not positive, then it is negative."
  8. "If a linear programming problem has a solution, it will have a corner point solution. This one has a solution, thus it has a corner point solution."
Solution

The arugments are:

  1. Disjunctive Syllogism.
  2. Modus Tollens.
  3. Reasoning by Transitivity.
  4. Modus Ponens.
  5. Modus Tollens
  6. Reasoning by Transitivity.
  7. Disjunctive Syllogism.
  8. Modus Ponens.

Subsection 4.5.2 Fallacies

The following are common Fallacies. They seem like they could be legitimate arguments, and are often employed, but they are not.

The first fallacy we discuss is the Fallacy of the Converse. The Fallacy of the Converse takes an implication \(p\to q\) and then argues that if \(q\) is true than \(p\) must be true as well.

\begin{equation*} \begin{array}{l} p\to q \\ q\\ \hline p \end{array} \end{equation*}

So for example "If I sell this TV, I will have $200. I have $200, so I must have sold the TV." The reason this is a fallacy is because, absent more information, you could have acquired $200 in any other way, it was never suggested that this is your only avenue to have acquired this money. Recall that both \(F\to T, T\to T\) are true statements, so knowing that \(q\) is true, does not allow you to conclude automatically that \(p\) is true.

Example 4.5.6.

The following are also examples of Fallacy of Converse:

  1. "If I speed, I'll get pulled over. I got pulled over, it must be for speeding."
  2. "If the crime stops, then there will be no reported cases. We stopped reporting cases, so crime must have stopped."
  3. "If \(S\) is a square, then \(S\) is a rectangle. \(S\) is a rectangle, thus \(S\) must be a square."

The next fallacy is the Fallacy of the Inverse. This supposes that you have an implication \(p\to q\) and then argues tht if \(p\) is false, than \(q\) must be false as well.

\begin{equation*} \begin{array}{l} p\to q \\ \sim p\\ \hline \sim q \end{array} \end{equation*}

So for example "If I order pizza, then I will have lunch. I didn't order pizza, I must not have had lunch." The reason this is a fallacy is because it is clear that you could potentially have had anything else for lunch. Without more knowledge, it can't be concluded based on this alone that you had no lunch options. Recall that \(F\to q\) is always true no matter what \(q\) is. So if you didn't get pizza, could have had lunch? Yes. Could you not have had lunch? Also yes. You simply can't conclude anything.

Example 4.5.7.

The following are also examples of Fallacy of the Inverse:

  1. "If I am hungry, then I will eat lunch. I'm not hungry, so I won't eat lunch."
  2. "If profits don't go up, people will get fired. Profits did go up, so no one will get fired."
  3. "If \((x,y)\) is an optimal solutin to a linear programming problem, then it is a corner point. \((x,y)\) is not an optimal solution, so it is not a corner point."
Checkpoint 4.5.8.

Identify if the argument is valid, and the argument or fallacy type for each of the following:

  1. "If there's an accident, traffic will be bad. Traffic wasn't bad, so there was no accident."
  2. "If someone eats rotten food, they will get sick. Someone got sick so they must have eaten rotten food."
  3. "If \(x\) is a multiple of 4, then it is not prime. Since \(x\) is not a multiple of 4, then it must be prime."
  4. "If \(\ell(x)\) is a linear function, than it has a slope. Since \(\ell(x)\) is a linear function, we can conclude it must have a slope."
Solution

The arugments are:

  1. Modus Tollens
  2. Fallacy of Converse.
  3. Fallacy of Inverse.
  4. Modus Ponens.

Subsection 4.5.3 Valid Arguments

Here, we will use our examples and fallacies above to construct examples of arguments, both valid and invalid.

Consider the following: If I fill the lawnmower with gas, and change the oil, it will start. I didn't start and I filled us the gas. Therefore I didn't change the oil.

So let:

  • \(g=\) “fill the lawnmmower with gas.”

  • \(o=\) “change the oil.”

  • \(s=\) “lawnmower starts.”

So what we know is \((g\wedge o)\to s, g \sim s\text{.}\) So:

\begin{equation*} \begin{array}{lcl} (g\wedge o)\to s\amp \ \ \ \amp \\ \sim s\\ \hline \sim(g\wedge o)=\sim g \vee\sim o \amp \amp \text{ Modus Tollens } \\ \\ \sim g \vee \sim o \\ g=\sim (\sim g)\\ \hline \sim o \amp \amp \text{ Disjunctive Syllogism } \end{array} \end{equation*}

Therefore \(\sim o\text{:}\) "I did not change the oil". This was a valid argument.

Next we give an example of a flawed argument.

Consider the following: If I drink coffee, I will be happy. If I drink coffee, I will be wired. If I am happy, then I am wired.

So let:

  • \(c=\) “drink coffee.”

  • \(h=\) “be happy.”

  • \(w=\) “be wired.”

What we have is \(c\to h, c\to w\text{.}\) But there's nothing here to connect \(h\) to \(w\text{.}\) If we knew this person drank coffee, we could make that conclusion, but since we don't know that, if we knew that they were happy, it could be for some reason other than drinking coffee. So we have no idea whether or not they'd be wired as well. So we cannot conclude \(h \to w\) and this is NOT a valid argument.

Checkpoint 4.5.9.

Suppose you knew the following:

If \(A, B\) are the same point, then the distance between them is zero. If \(A, B\) are different points, there is exactly one line containing both \(A\) and \(B\text{.}\) There is not exactly one line containing both \(A\) and \(B\text{.}\)

Construct a valid argument to show the distance betweem \(A\) and \(B\) is zero.

Solution

Let's define the following:

  • \(s=\)"\(A, B\) are the same point."
  • \(z=\)"The distance between \(A, B\) is zero."
  • \(\ell=\)"There is exactly one line cointaining both \(A\) and \(B\text{.}\)"

Then what we are given is:

  • \(\displaystyle s\to z\)
  • \(\displaystyle \sim s \to \ell\)
  • \(\displaystyle \sim\ell\)

Thus we can proceed as follows:

\begin{equation*} \begin{array}{lcl} \sim s\to \ell\amp \ \ \ \amp \\ \sim \ell \\ \hline \sim (\sim s)=s \amp \amp \text{ Modus Tollens } \\ \\ s\to z \\ s\\ \hline z \amp \amp \text{ Modus Ponnens } \end{array} \end{equation*}

Thus concluding \(z=\)"The distance between \(A, B\) is zero."

Checkpoint 4.5.10.

Suppose you knew the following:

If Frida has breakfast, she will have either pancakes or waffles. Frida had breakfast and did not have waffles.

Construct a valid argument to show Frida had pancakes.

Solution

Let's define the following:

  • \(b=\)"Frida had breakfast."
  • \(p=\)"Frida had pancakes."
  • \(w=\)"Frida had waffles."

Then what we are given is:

  • \(\displaystyle b\to (p\vee w)\)
  • \(\displaystyle b\)
  • \(\displaystyle \sim w\)

Thus we can proceed as follows:

\begin{equation*} \begin{array}{lcl} b\to (p\vee w)\amp \ \ \ \amp \\ b \\ \hline p\vee w \amp \amp \text{ Modus Ponnens } \\ \\ p\vee w \\ \sim w\\ \hline p \amp \amp \text{ Disjunctive Syllogism } \end{array} \end{equation*}

Thus concluding \(p=\)"Frida had pancakes."

Checkpoint 4.5.11.

Find the flaw in the following line of reasoning:

If \(x\)is a multiple of 5 and a multiple of 2, then \(x\) is a multiple of 10. \(x\) is not a multiple of 10 or 2, so it must be a multiple of 5.

Solution

Let's define the following:

  • \(t=\)"\(x\) is a multiple of ten."
  • \(f=\)"\(x\) is a multiple of five."
  • \(e=\)"\(x\) is a multiple of two."

Then what we are given is:

  • \(\displaystyle f\wedge e\to t\)
  • \(\displaystyle \sim t\)
  • \(\displaystyle \sim e\)

So one hypothetically start with Modus Tollens:

\begin{equation*} \begin{array}{lcl} (f\wedge e)\to t\amp \ \ \ \amp \\ \sim t \\ \hline \sim(f\wedge e)=\sim f \vee\sim e\amp \amp \text{ Modus Tollens } \end{array} \end{equation*}

Next we are given \(\sim e\text{.}\) But from this we cannot use Disjunctive Syllogism to conclude \(\sim (\sim f)=f\text{.}\) Notice that in an or statement, both statements can be true. So if \(x\) isn't a multiple of two, it doesn't have to be a multiple of five. 7 is neither a multiple of 2, 5 nor 10.

Checkpoint 4.5.12.

Find the flaw in the following line of reasoning:

If it snows, the city will send out snow plows. If the city sends out snow plows, traffic will get bad. Traffic got bad, so therefore it snowed.

Solution

Let's define the following:

  • \(s=\)"It snows."
  • \(p=\)"The city sends out snow plows."
  • \(t=\)"Traffic got bad."

Then what we are given is:

  • \(\displaystyle s\to p\)
  • \(\displaystyle p\to t\)
  • \(\displaystyle t\)

So one hypothetically start with Reasoning by Transitivity:

\begin{equation*} \begin{array}{lcl} s\to p\amp \ \ \ \amp \\ p\to t \\ \hline s\to t\amp \amp \text{ Reasnoning by Transitivity } \end{array} \end{equation*}

Next we are given \(t\text{.}\) But we cannot from \(t\) conclude \(s\text{.}\) This would be the Fallacy of the Converse. There's all sorts of reasons traffic might be bad, it may have nothing to do with snows or snow plows.
