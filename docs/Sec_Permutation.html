Section 9.1 Multiplication Principle and Permutations

In this section, we count the number of ways to make selections in order.

Subsection 9.1.1 Multiplication Principle

If you were eating breakfast, and you could choose from eggs, sausage or pancakes to eat, and coffee and juice to drink, how many ways could you pick one food item and one beverage?

It's clear, we have 3 choice for food items say

\begin{equation*} \color{red}{E}, \color{blue}{S}, \color{teal}{P}. \end{equation*}

Then for each choice of beverage, we have two options for each of our original options

\begin{equation*} \color{red}{E}C, \color{red}{E}\color{orange}{J}, \color{blue}{S}C, \color{blue}{S}\color{orange}{J}, \color{teal}{P}C, \color{teal}{P}\color{orange}{J}. \end{equation*}

Which leaves us with \(3\cdot 2=6\) options. If we have 5 choices for food, and 7 choices for beverage, we can see that we would have \(5\cdot 7=35\) choices. If we then had another 4 choices for eggs, and were picking a main item, a beverage, and eggs, that would leave \(5\cdot 7\cdot 4=140\) choices. This leads to the following:

Remark 9.1.1. Multiplication Principle.
If we have to make \(n\) choices, each with \(m_1, m_2, m_3,\ldots,m_n\) choices each, then the number of ways we can pick one of each choice is
\begin{equation*} m_1\cdot m_2\cdots m_n. \end{equation*}
Example 9.1.2.

Suppose you had to make a 5 letter passcode using the 26 letters in the English alphabet (all lowercase).

  1. How many ways can this be done if you can repeat letters?
  2. If you can't?
Solution

Using the multiplication principle:

  1. If you pick 5 times with 26 choices each, that is \(26\cdot26\cdot26\cdot26\cdot26=26^5\) choices.
  2. Each time you pick a letter, you reduce the number of choices for the next letter, so you have 26 choices for the first letter, then 25 for the next, then 24 etc. so \(26\cdot25\cdot24\cdot23\cdot 22\) choices.
Checkpoint 9.1.3.

If you flip a coin 3 times, how many different head, tails arrangements are possible?

Solution

Since each coin flip has two outcomes, there are \(2\cdot 2\cdot 2=2^3=8\) possible outcomes.

\begin{equation*} HHH, HHT, HTH, THH, TTH, THT, TTH, TTT. \end{equation*}
Checkpoint 9.1.4.

If a resturaunt offered 4 different appetizers, 7 main courses and 5 desserts, how many ways can you order one of each?

Solution

There would be \(4\cdot 7\cdot 5=140\) different possible orders.

Subsection 9.1.2 Permutations

The sort of situation described in the second part of Example 9.1.2 is called a permutation.

Definition 9.1.5.

A permutation is an ordered arrangement of objects. In partcular, a \(r\) permutation on an \(n\) element set is a selection of \(r \) objects from this set and arranged.

The number of \(r\) permutations on an \(n\) element set would be computed

\begin{equation*} P(n,r)=\overbrace{n\cdot (n-1)\cdots(n-r+1)}^r. \end{equation*}

So the number of ways to take the letters \(A, B, C, D, E \) pick 3 and arrange them is \(P(5,3)=5\cdot 4\cdot 3=60\text{:}\)

\begin{equation*} ABC, ACB, BAC, BCA, CAB, CBA, ABD, ADB, BAD, BDA, DAB, DBA, \end{equation*}
\begin{equation*} ABE, AEB, BAE, BEA, EAB, EBA, ACD, ADC, CAD, CDA, DAC, DCA, \end{equation*}
\begin{equation*} ACE, AEC, CAE, CEA, EAC, ECA, ADE, AED, DAE, DEA, EAD, EDA, \end{equation*}
\begin{equation*} BCD, BDC, CBD, CDB, DBC, DCB, BCE, BEC, CBE, CEB, EBC, ECB, \end{equation*}
\begin{equation*} BDE, BED, DBE, DEB, EBD, EDB, CDE, CED, DCE, DEC, ECD, EDC. \end{equation*}

Note that

\begin{equation*} 5\cdot 4\cdot 3=\frac{5\cdot 4\cdot 3\cdot \color{blue}{2}\cdot \color{blue}{1}}{\color{blue}{2}\cdot \color{blue}{1}}. \end{equation*}

This leads us to:

Remark 9.1.6.

We note that \(n!\) denotes \(n!=\overbrace{n\cdot (n-1)\cdots 1}^{n}\) where we let \(0!=1\text{.}\)

Notice that

\begin{equation*} P(n,n)=\overbrace{n\cdot (n-1)\cdots 1}^{n}=n!. \end{equation*}

Also notice that more generally

\begin{equation*} P(n,r)=\overbrace{n\cdot (n-1)\cdots (n-r+1)}^{r}=\frac{n\cdot (n-1)\cdots (n-r+1)\cdot \color{blue}{(n-r)\cdots 1}}{\color{blue}{(n-r)\cdots 1}}=\frac{n!}{(n-r)!}. \end{equation*}
Example 9.1.7.

If you have 10 books, how many ways can you select 6 of them to arrange on a shelf?

Solution

This would be

\begin{equation*} P(10,6)=\frac{10!}{4!}=10\cdot 9\cdot 8\cdot 7\cdot 6\cdot 5=151200. \end{equation*}

Checkpoint 9.1.8.

If a driver had to make 6 deliveries, how many ways could they pick an order to do so?

Solution

This would be

\begin{equation*} P(6,6)=6!= 6\cdot 5\cdot 4\cdot 3\cdot 2\cdot 1=72- \end{equation*}
Checkpoint 9.1.9.

If a class of 30 had to select 4 students to be class president, vice president, secretary and treasurer?

Solution

This would be

\begin{equation*} P(30,4)=\frac{30!}{26!}= 30\cdot 29\cdot 28\cdot 27=657720- \end{equation*}

Subsection 9.1.3 Repeated Objects

How many ways are there to take

\begin{equation*} A, B, B, B \end{equation*}

and arrange them? If the B's were different, there would be \(4!=24\) ways to do so:

\begin{equation*} \begin{array}{cccc} AB_1B_2B_3 \amp B_1AB_2B_3 \amp B_1B_2AB_3 \amp B_1B_2B_3A\\ AB_1B_3B_2 \amp B_1AB_3B_2 \amp B_1B_3AB_2 \amp B_1B_3B_2A\\ AB_2B_1B_3 \amp B_2AB_1B_3 \amp B_2B_1AB_3 \amp B_2B_1B_3A\\ AB_2B_3B_1 \amp B_2AB_3B_1 \amp B_2B_3AB_1 \amp B_2B_3B_1A\\ AB_3B_1B_2 \amp B_3AB_1B_2 \amp B_3B_1AB_2 \amp B_3B_1B_2A\\ AB_3B_2B_1 \amp B_3AB_2B_1 \amp B_3B_2AB_1 \amp B_3B_2B_1A \end{array} \end{equation*}

In the above arrangement, we have all the \(B\)'s in the same “position”. Thus of we remove the disntinction between the \(B\)'s, we are left with just the \(\frac{4!}{3!}=4\) arrangements

\begin{equation*} ABBB, BABB, BBAB, BBBA. \end{equation*}

the \(4!\) is the number of arrangements with distinction, and the \(3!\) the number of ways to arrange the B's.

Remark 9.1.10.

The number of ways to arrange \(n\) objects, where \(r\) of them are repeated \(m_1, \ldots, m_r\) each is

\begin{equation*} \frac{n!}{m_1!\cdots m_r!} \end{equation*}

There are \(n!\) ways to arrange these objects with distinction, and \(m_1!\cdots m_r!\) removes those distinctions.

Example 9.1.11.

How many ways can you arrange the letters of MISSISSIPPI?

Solution

There are 11 letters in MISSISSIPPI, 4 I's, 4 S's and 2 P's. So the number of ways to arrange these letters is

\begin{equation*} \frac{11!}{4!4!2!}=34650. \end{equation*}
Checkpoint 9.1.12.

A student buys 3 strawberry yogurts, 2 raspberry yogurts, and 2 blueberry yogurts. They put them in their dormitory refrigerator to eat one a day for the next week. Assuming yogurts of the same flavor are indistinguishable, in how many ways can they select yogurts to eat for the next week?

Solution

There are 7 yogurts: 3 strawberry yogurts, 2 raspberry yogurts, and 2 blueberry yogurts. So the number of ways to arrange these yohurts is

\begin{equation*} \frac{7!}{3!2!2!}=210. \end{equation*}
