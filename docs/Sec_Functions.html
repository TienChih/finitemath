Section 6.1 Introduction to Functions

In this section, we introduce the definition of a function, function notation, and some properties that functiond have.

Subsection 6.1.1 What is a function?

Definition 6.1.1.

A function \(f\) is a rule that assigns to each element from one set exactly one element from another set.

The set of all possible inputs is called the domain, and the set of all possible outputs is the range.

For any \(x\) in the domain, \(f(x)\) denotes the appropriate output in the range. So in the expression say

\begin{equation*} f(a)=b, \end{equation*}

we have that \(a\) is an element of the domain, and \(f(a), b\) an element of the range.

Note that in general, neither the domain nor range need neccesarily be collections of numbers, and that any set is potentially a domain or range.

In the above diagram, we can see that this is a function. The domain is \(\{\text{Andrew, Brianna, Connie, Deng}\}\text{,}\) the range is \(\{\text{Chocolate, Mint}\}\text{.}\) Avacado is not in the range, since no element of the domain evaluates to Avacado. If we name the function \(\alpha\text{,}\) we would have \(\alpha(\text{Andrew})=\text{Chocolate}=\alpha(\text{Brianna})\) and \(\alpha(\text{Connie})=\text{Mint}=\alpha(\text{Deng})\text{.}\)

In this case, we can see this relation is not a function. Let's call it \(L\) for lunch. We see that many of our subjects had more than one item for lunch. So If we asked about \(L(\text{Andrew})\text{,}\) it's not clear if this should be Hamburger or Fries, or a Soda. Recall that the definition of a function assigns one output per input.

Checkpoint 6.1.4.

Which of the following assignments or relations are functions?

  1. To each student \(S\text{,}\) we assign to them their student id number.
  2. To each person \(P\text{,}\) we let \(N(P)\) be the name of \(P\)'s' parent.
  3. We assign \(x\) to \(a(x)\) as follows:
    \begin{equation*} \begin{array}{|c||c|c|c|c|c|} \hline x \amp 1\amp 2\amp 3\amp 4\amp 5\\ \hline a(x)\amp 3\amp 2\amp 3\amp -27\amp 3\\ \hline \end{array} \end{equation*}
  4. We assign \(x\) to \(b(x)\) as follows:
    \begin{equation*} \begin{array}{|c||c|c|c|c|c|} \hline x \amp 1\amp 2\amp 2\amp 3\amp 3\\ \hline b(x)\amp 6\amp 7\amp 8\amp 9\amp 10\\ \hline \end{array} \end{equation*}
Solution
  1. This is a function, since each student has one and only one student id number.
  2. This is not a function, since there is not a unique parent for each person, and thus potentially more than on assignment.
  3. This is a function because each input is assigned one output.
  4. This is not a function because inputs 2 and 3 are assigned more than one output.

Subsection 6.1.2 Real valued functions

In this course, we will focus on real-valued functions, where the inputs and outputs are real numbers. These are typically defined algebraiacally where the inputs are numbers, and the outputs are algebraic operations performed on those numbers.

Consider \(f(x)=4-x^2\text{.}\) This is an assignment that assigns to each real number \(x\) a real number \(4-x^2\text{:}\)

  • \(f(0)=4-0^2=4\text{.}\)
  • \(f(2)=4-2^2=0\text{.}\)
  • \(f(-1)=4-(-1)^2=3\text{.}\)
  • \(f(10)=4-10^2=-96\text{.}\)
  • \(f(a)=4-a^2\text{.}\)

We can see in the Desmos graph below that by adjusting \(a\text{,}\) we may view all the potential inputs and outputs of \(f\text{.}\)

Sometimes we express real valued functions as an algebraic relation between an independent variable (typically \(x\) but not necissarily) as an input, and a dependent vairable (typically \(y\) but not necissarily) as an output. To check to see if a relation is a function, we often employ the vertical line test: we check to see that for each vertical line, it intersects the curve of the relation at most once, that is, each input only has one output.

We can see that the relation \(y=\sqrt{10-x}\) expresses \(y\) as a function of \(x\text{:}\) Each valid input of \(x\) has only one output. However, the relation \(y^2+x^2=25\text{:}\) as several inputs generate two outputs.

Checkpoint 6.1.5.

Which of the following relations express \(y\) as a function of \(x\text{?}\)

  1. \(y=x^2+x\text{.}\)
  2. \(x=1-y^2\text{.}\)
  3. \(y^2=x+2\text{.}\)
  4. \(x=y+3\text{.}\)
Solution

Consider the following:

  1. This is a function, since each \(x\) only has one associated one \(y\) value.
  2. This is not a function, since there is not a unique \(y\) for each \(x\text{.}\) So for example, when \(x=1\text{,}\) we have that \(y=1, -1\) both satisfy this relation.
  3. This is not a function, since there is not a unique \(y\) for each \(x\text{.}\) So for example, when \(x=2\text{,}\) we have that \(y=2, -2\) both satisfy this relation.
  4. This is a function, since each \(x\) only has one associated one \(y\) value.

We can see the visualizations of these curves above, and see which satisfy the "vertical line test".

Subsection 6.1.3 Domain and Range

Definition 6.1.6.

An interval is the collection of all real numbers subject to certain bounds:

  • \([a,b]\) represents all real numbers \(x\) where \(a\leq x\leq b\text{.}\)
  • \([a,b)\) represents all real numbers \(x\) where \(a\leq x \lt b \text{.}\)
  • \((a,b]\) represents all real numbers \(x\) where \(a\lt x\leq b\text{.}\)
  • \((a,b)\) represents all real numbers \(x\) where \(a\lt x \lt b \text{.}\)

In other words \([,]\) include the endpoints and \((,)\) exclude them. Note that we may use \(\infty, -\infty\) as endpoints, but they cannot be included as they are not real numbers.

Generally speaking for real valued functions, any real number may be an element of the domain, unless otherwise specified, or unless it would result in an undefined arithmetic operation.

Example 6.1.7.

Consider the domain and range of \(f(x)=x^2-2\text{.}\) No matter what the value of \(x\) is, that value may be squared, and 2 may be subtracted from the result. Thus the domain is \((-\infty, \infty)\text{,}\) every real number is a valid input.

Note that \(x^2\) has minimal value 0, so \(x^2-2\geq -2\) and the range of \(f(x)\) is \([-2,\infty)\text{,}\) numbers greater than or equal to 2 are valid outputs.

Example 6.1.8.

Consider the domain and range of \(g(x)=\frac{2}{3-x} \text{.}\) In this case, we note that 2 divided by any non-zero number is well-defined. However, we cannot divide two by zero. Thus the domain is \((-\infty, 3)\cup (3,\infty)\text{,}\) every real number other than 3 is a valid input, since \(\frac{2}{3-3}=\frac{2}{0}\) is undefined.

Similarly, there is no value we may divide 2 by to achieve 0, and thus the range is \((-\infty, 0)\cup (0,\infty)\text{,}\) every real value other than zero is a valid output.

Example 6.1.9.

Consider the domain and range of \(h(x)=\sqrt{2+x} \text{.}\) In this case, we note that we may take the squareroot of zero, but not negative numbers. We note then that \(2+x\geq0\) for \(h(x)\) to be defined. Thus the domain is \((-2,\infty)\text{,}\) every real number greater than or equal to -2 is a valid input.

Similarly, the output of the square root function is non-negative, and thus the range is \([0,\infty)\text{,}\) every real value greater than or equal to zero is a valid output.

Checkpoint 6.1.10.

Find the domain and range of \(a(x)=x^3-2\text{.}\)

Solution

Both the domain and range are \((-\infty, \infty)\text{.}\)

Checkpoint 6.1.11.

Find the domain and range of \(b(x)=\sqrt{x^2-4}\text{.}\)

Solution

Note that \(b(x)\) is defined so long as \(x^2-4\geq 0\) which occurs when \(x\leq -2\) and \(x\geq 2\text{.}\) Thus the domain is \((-\infty, -2)\cup (2, \infty)\text{.}\) Since the output of the radical function is non-negative, we have that the domain is \([0,\infty)\text{.}\)

Checkpoint 6.1.12.

Find the domain and range of \(c(x)=\frac{1}{\sqrt{x}}+2\text{.}\)

Solution

Note that \(c(x)\) is defined so long as \(\sqrt{x}\neq 0\) and is defined, which occurs when \(x>0\text{.}\) Thus the domain is \((0, \infty)\text{.}\) Since the output of the radical function is non-negative, so are the outputs of \(\frac{1}{\sqrt{x}}\text{,}\) excluding zero since that is not possible. Then when we add 1, we have that the domain is \((1,\infty)\text{.}\)

Checkpoint 6.1.13.

Find the range of \(d(x)=x^2\) restricted to the domain \(\{-3,-2,1,2\}\text{.}\)

Solution

Note then that \(d(-3)=9, d(-2)=4, d(1)=1, d(2)=4\text{.}\) So the range is \(\{1,4,9\}\text{.}\)

Subsection 6.1.4 Evaluating functions

The algebraic definition of real valued function is a guide on how they are to be evaluated. It may be that we will wish to evaluate these at real number values, but it is possible we will want to evluate them at algebraic expressions themselves.

Example 6.1.14.

Consider \(f(x)=x^2+x-1\text{.}\) Evaluate \(f(x)\) as follows:

  1. \(f(-2)\text{.}\)
  2. \(f(3)\text{.}\)
  3. \(f(a)\text{.}\)
  4. \(f(2x)\text{.}\)
  5. \(f(x+1)\text{.}\)
  6. \(f(x+h)\text{.}\)
Solution

We evaluate:

  1. \(f(-2)=(-2)^2+(-2)+1=4-2+1=3\text{.}\)
  2. \(f(3)=3^2+3-1=11\text{.}\)
  3. \(f(a)=a^2+a-1\text{.}\)
  4. \(f(2x)=(2x)^2+(2x)-1=4x^2+2x-1\text{.}\)
  5. \(f(x+1)=(x+h)^2+(x+h)-1=x^2+2hx+h^2+x+h-1\text{.}\)
Checkpoint 6.1.15.

Consider \(g(x)=1+\sqrt{x+1}\text{.}\) Evaluate \(f(x)\) as follows:

  1. \(g(0)\text{.}\)
  2. \(g(-3)\text{.}\)
  3. \(g(x^2)\text{.}\)
  4. \(g(a+b)\text{.}\)
Solution

We evaluate:

  1. \(g(0)=1+\sqrt{0+1}=2\text{.}\)
  2. \(g(-3)=1+\sqrt{-3+1}\) which is undefined.
  3. \(g(x^2)=1+\sqrt{x^2+1}\text{.}\)
  4. \(g(a+b)=1+\sqrt{a+b+1}\text{.}\)
Checkpoint 6.1.16.

Consider \(h(x)=\frac{2}{x+3}\text{.}\) Evaluate \(f(x)\) as follows:

  1. \(h(4)\text{.}\)
  2. \(h(-1)\text{.}\)
  3. \(h(x-1)\text{.}\)
  4. \(h(x+h)\text{.}\)
Solution

We evaluate:

  1. \(h(4)=\frac{2}{4+3}=\frac{2}{7}\text{.}\)
  2. \(h(-1)\frac{2}{-1+3}=\frac{2}{2}=1\text{.}\)
  3. \(h(x-1)=\frac{2}{x-1+3}=\frac{2}{x+2}\text{.}\)
  4. \(h(x+h)=\frac{2}{x+h+3}\text{.}\)
Checkpoint 6.1.17.

Suppose that the cost of producing \(x\) units of a good is \(C(x)=\$20x+100\text{.}\) :

  1. What is the cost of producing 30 units?
  2. Suppose that the rate of production of this good increases so that in week \(t\) of production, you produce \(t^2+10\) units of this good. What is the cost of production in week \(t\text{?}\)
Solution

We evaluate:

  1. This would be \(C(30)=20\cdot30+100=\$700\text{.}\)
  2. In week \(t\) you produce \(t^2+10\) units, so the cost would be \(C(t^2+10)=20\cdot(t^2+10)+100=\$(20t^2+300)\text{.}\)
