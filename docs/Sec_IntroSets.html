Section 8.1 Introduction to Sets

Sets are a formal mathematical way of describing collections of things or objects. The actual “order” objects appear, or repition of objects do not matter here. Set's fundamentally are collections of things, where any object may be described as in it, or not in it.

Subsection 8.1.1 Set Basics

We often denote sets as actual lists:

\begin{equation*} \{1,2,3,4,5\} \end{equation*}

or in terms of a condition:

\begin{equation*} \{x\ \text{a natural number}:1\leq x \leq 5\}. \end{equation*}

The above should read “All natural numbers \(x\) where \(x\) is between 1 and 5.” That is, the set \(\{1,2,3,4,5\}\text{.}\)

Definition 8.1.1.

We say that an element \(a\) is an element of \(A\) if \(a\) belongs to \(A\text{.}\) We denote this \(x\in A\text{.}\) Otherwise, \(a\) is not an element of \(A\) and \(a\not\in A\text{.}\)

Example 8.1.2.

Consider the set \(A=\{2,3,5\}.\)

  • \(1\not\in A.\) since 1 is not in \(A\text{.}\)
  • \(2\in A.\) since 2 is in \(A\text{.}\)
  • \(3\in A.\) since 3 is in \(A\text{.}\)
  • \(4\not\in A.\) since 4 is not in \(A\text{.}\)
  • \(5\in A.\) since 5 is in \(A\text{.}\)
Example 8.1.3.

Consider the set \(S=\{x:x\ \text{is a word starting with an "A"}\}.\)

  • Apple, Agnostic, Aggregate, Anchovies \(\in S\text{.}\)
  • Bannana, Xenomorph, Cumble, Textual, Finite, Mathematics \(\not\in S\text{.}\)
For this chapter, the primary goal of sets is to list a collection of possible outcomes, or outcomes which satify certain requirements. It may be that there may be situations where no outcomes satisfy the requirements. To describe such scenarios, we require a set with no elements.
Definition 8.1.4.

The set with no elements is the empty set or null set denoted \(\emptyset\text{.}\)

Example 8.1.5.

Consider the set

\begin{equation*} R=\{a:a^2-2a=0\ \text{and}\ a\ \text{is odd}\}. \end{equation*}

We note that \(a^2-2a=0\) has two solutions, \(a=0,2\text{.}\) But they are both even, neither are odd. Thus \(R=\emptyset\text{.}\)

There are situations where every element which satisfies a condition also satisfies other conditions. We then need to denote when the elements of a set also belong to another set.
Definition 8.1.6.

We say \(A\) is a subset of \(B\) if each element of \(A\) is also an element of \(B\text{,}\) denoted \(A\subseteq B\text{.}\) If \(A\) contains an element not in \(B\text{,}\) we say \(A\not\subseteq B\text{.}\)

Remark 8.1.7.

Given any set \(A\text{:}\)

  • \(A\subseteq A\) since each element of \(A\) is definitely an element of \(A\text{.}\)
  • \(\emptyset\subseteq A\) since all none elements of \(\emptyset\) are also elements of \(A\text{!}\)
Example 8.1.8.

Consider the sets

\begin{equation*} A=\{1,2,3,4,5\}, B=\{1,3,5\}, C=\{1,4\}. \end{equation*}
  • \(B\subseteq A\) since each element of \(B\) is an element of \(A\text{.}\)
  • \(C\subseteq A\) since each element of \(C\) is an element of \(A\text{.}\)
  • \(B\not\subseteq C\) since \(3,5\in B\) but \(3,5\not\in C\text{.}\)
  • \(C\not\subseteq B\) since \(4\in C\) but \(4\not\in B\text{.}\)
  • \(\emptyset \subseteq A, B, C, \emptyset\text{.}\)
Definition 8.1.9.

We say sets \(A, B\) are equal if each element of \(A\) is also an element of \(B\text{,}\) and each element of \(B\) is also an element of \(A\text{.}\) This is denoted \(A=B\text{.}\) Otherwise \(A\neq B\text{.}\)

Note that \(A=B\text{.}\) is equivalent to sayng \(A\subseteq B, B\subseteq A\text{.}\)

Example 8.1.10.

Consider the sets

\begin{equation*} V=\{4,1,2\}, W=\{4,2\}, X=\{2,4\}, Y=\{1,2,4,2\}, Z=\{a:x^2-6x+8=0\}. \end{equation*}
  • \(V\neq W\text{,}\) since \(1\not\in W\)
  • \(V=Y\) since each element of \(V\) is an element of \(Y\) and vice versa (one can check them one by one).
  • \(W=X=Z\) since they all only contain \(2, 4\text{.}\)

Subsection 8.1.2 Set Operations

Generally speaking, while dealing with sets, there is an understood universal set in which we are working. In a probability setting, this usually denotes the set of all possible outcomes of an experiment. For now, we'll just call this set \(\mathcal{U}\text{.}\)
Definition 8.1.11.

Given a set \(A\) we denote the complement of \(A\) to be the set \(A^c\) consisting of all elements of \(\mathcal{U}\) not in \(A.\)

Example 8.1.12.

Let

\begin{equation*} \mathcal{U}=\{Ailca, Bryan, Corrine, Damien, Esmeralda, Francis\}. \end{equation*}

Let \(A=\{Bryan, Esmeralda\}, B=\{Bryan, Francis\}\text{.}\)

  • \(A^c=\{Alicia, Corrine, Damien, Franics\}\text{,}\) all the elements of \(\mathcal{U}\) that aren't in \(A\text{.}\)
  • \(B^c=\{Ailca,Corrine, Damien, Esmeralda\}\text{,}\) all the elements of \(\mathcal{U}\) that aren't in \(B\text{.}\)
  • \(\emptyset^c=\mathcal{U}=\{Ailca, Bryan, Corrine, Damien, Esmeralda, Francis\}\) since each element of \(\mathcal{U}\) is not contained in the emptyset.
Remark 8.1.13.

Given any set \(A\text{,}\) and universal set \(\mathcal{U}\text{:}\)

  • \((A^c)^c=A\text{.}\)
  • \(\emptyset^c=\mathcal{U}\text{.}\)
  • \(\mathcal{U}^c=\emptyset\text{.}\)
Definition 8.1.14.

Given a sets \(A, B\text{:}\)

  • The union of \(A\) and \(B\) denoted \(A\cup B\) consists of all elements that are ether in \(A\) or \(B\text{.}\)
  • The intersection of \(A\) and \(B\) denoted \(A\cap B\) consists of all elements that are both in \(A\) and \(B\text{.}\)
Example 8.1.15.

Recall Example 8.1.12. Find the following:

  • \(A\cup B\text{.}\)
  • \(A\cap B\text{.}\)
  • \(A^c\cup B\text{.}\)
  • \(A\cap B^c\text{.}\)
Solution

Recall that

\begin{equation*} A=\{Bryan, Esmeralda\}, B=\{Bryan, Francis\}, \end{equation*}
\begin{equation*} A^c=\{Alicia, Corrine, Damien, Franics\}, B^c=\{Ailca,Corrine, Damien, Esmeralda\}. \end{equation*}
  • \(A\cup B=\{Bryan, Esmeralda, Francis\}\text{.}\)
  • \(A\cap B=\{Bryan\}\text{.}\)
  • \(A^c\cup B=\{Ailca, Bryan, Corrine, Damien, Francis\}\text{.}\)
  • \(A\cap B^c=\{Esmeralda\}\text{.}\)
Remark 8.1.16.

Given any set \(A\text{:}\)

  • \(A\cup A^c=\mathcal{U}\text{.}\)
  • \(A\cap A^c=\emptyset\text{.}\)

Theorem 8.1.21.

Given sets \(A, B\text{:}\)

  • \((A\cup B)^c=A^c\cap B^c\text{.}\)
  • \((A\cap B)^c=A^c\cup B^c\text{.}\)
