Section 2.2 Representing a System of Linear Equations as a Matrix

Given a system of equations, we can try to encode that information in a matrix:

So for example the system of equations:

\begin{align*} 2x+3y+z\amp = 3\\ -2x+4y+2z\amp = 0\\ x-y+2z\amp = 3 \end{align*}

Would have matrix representation:

\begin{equation*} \left( \begin{array}{rrr|r} 2 \amp 3 \amp 1\amp 3\\ -2 \amp 4 \amp 2 \amp 0\\ 1 \amp -1 \amp 2 \amp 3 \end{array} \right)\text{.} \end{equation*}

A matrix is a rectangular array of numbers, each number is an entry of the matrix. In the above example, the vertical line is a visual reminder to separate the constant terms from the variables. A matrix displayed this way is an augmented matrix.

By letting each column of a matrix represent a different variable or constant and arranging the linear equations into rows, one can write any linear system of equations as an augmented matrix. Conversely, if you have a matrix:

\begin{equation*} \left( \begin{array}{rr|r} 1 \amp 2 \amp 4\\ 3 \amp 3 \amp 9 \end{array} \right) \end{equation*}

This represent the system of equations:

\begin{align*} x+2y\amp = 4\\ 3x+3y\amp = 9 \end{align*}

Subsection 2.2.1 Solving Systems of Linear Equations

So let's consider the above system/matrix:

\begin{equation*} \left( \begin{array}{rr|r} 1 \amp 2 \amp 4\\ 3 \amp 3 \amp 9 \end{array} \right) \end{equation*}

We are allowed to manipulate this matrix in a number of ways. Remembering that each row represents a linear equation:

  1. You can multiply any row by a non-zero multiple. We can do this because say, the solutions to \(x+2y=4\) are the same as the solutions to \(-x-2y=-4\) or \(2x+4y=8\text{,}\) so we're not adding in new solutions or removing existing solutions.

  2. You can add to any row, the multiple of any row. Adding a linear equality to another linear equality preserves the solutions to both equations.

  3. You can rearrange the rows. The solutions to a system of equations does not depend on the order in which they appear.

So our goal here is to change this system into one where it's much easier to see what the solution is. The best way to do this, is to not have all these x's and y's, I want 2 equations: \(x=?, y=??\text{.}\)

So:

\begin{align*} \left( \begin{array}{rr|r} 1 \amp 2 \amp 4\\ 3 \amp 3 \amp 9 \end{array} \right)\amp \amp\\ (1/3)R_2\mapsto R_2 \left( \begin{array}{rr|r} 1 \amp 2 \amp 4\\ 1 \amp 1 \amp 3 \end{array} \right)\amp \amp\\ (-1)R_1+R_2\mapsto R_2 \left( \begin{array}{rr|r} 1 \amp 2 \amp 4\\ 0 \amp -1 \amp -1 \end{array} \right)\amp \amp\\ (-1)R_2\mapsto R_2 \left( \begin{array}{rr|r} 1 \amp 2 \amp 4\\ 0 \amp 1 \amp 1 \end{array} \right)\amp \amp\\ (-2)R_2+R_1\mapsto R_1 \left( \begin{array}{rr|r} 1 \amp 0 \amp 2\\ 0 \amp 1 \amp 1 \end{array} \right)\amp \amp \end{align*}

What system of equations of this? It is:

\begin{align*} x+0y\amp = 2\\ 0x+y\amp = 1 \end{align*}

Which must mean: \(x=2, y=1\text{,}\) alright! By the way, one can test your answers to see if they are correct!

\begin{equation*} 1(2)+2(1)=4, 3(1)+3(2)=9 \end{equation*}

so it checks out.

A matrix with 1's across the diagonal, and 0's above and below these 1's (as above) is called reduced row-echelon form. A process to obtain a reduced row echelon form matrix is called the Gauss-Jordan Method. The methodology is:

  1. Starting with the first row, multiply by a number so it's leading (leftmost) value is 1.

  2. Multiply and add copies of this row to all rows below so that every entry below this leading 1 is a 0.

  3. Repeat steps (1) and (2) for subsequent rows until the last row.

  4. Starting with the bottom row working up, repeat steps (1)-(3) except working upwards and making entries above the leading 1's 0.

Example 2.2.1.

Solve the linear system:

\begin{align*} 3x+2y+z\amp = 1\\ x+y-z\amp = 3\\ 2x-y+2z\amp = -2 \end{align*}

This corresponds to augmented matrix:

\begin{equation*} \left( \begin{array}{rrr|r} 3 \amp 2 \amp 1 \amp 1\\ 1 \amp 1 \amp -1 \amp 3\\ 2 \amp -1 \amp 2 \amp -2 \end{array} \right) \end{equation*}

So applying Gauss-Jordan:

\begin{align*} \frac{1}{3}R_1\to R_1 \left( \begin{array}{rrr|r} 3 \amp 2 \amp 1 \amp 1\\ 1 \amp 1 \amp -1 \amp 3\\ 2 \amp -1 \amp 2 \amp -2 \end{array} \right)\amp \amp\\ \frac{1}{3}R_1\to R_1 \left( \begin{array}{rrr|r} 1 \amp \frac{2}{3} \amp \frac{1}{3} \amp \frac{1}{3}\\ 1 \amp 1 \amp -1 \amp 3\\ 2 \amp -1 \amp 2 \amp -2 \end{array} \right)\amp \amp\\ -1R_1+R_2\to R_2\left( \begin{array}{rrr|r} 1 \amp \frac{2}{3} \amp \frac{1}{3} \amp \frac{1}{3}\\ 0 \amp -\frac{1}{3} \amp -\frac{4}{3} \amp \frac{8}{3}\\ 2 \amp -1 \amp 2 \amp -2 \end{array} \right)\amp \amp\\ -3R_2\to R_2\left( \begin{array}{rrr|r} 1 \amp \frac{2}{3} \amp \frac{1}{3} \amp \frac{1}{3}\\ 0 \amp 1 \amp 4 \amp -8\\ 2 \amp -1 \amp 2 \amp -2 \end{array} \right)\amp \amp\\ -2R_1+R_3\to R_3\left( \begin{array}{rrr|r} 1 \amp \frac{2}{3} \amp \frac{1}{3} \amp \frac{1}{3}\\ 0 \amp 1 \amp 4 \amp -8\\ 0 \amp -\frac{7}{3} \amp \frac{4}{3} \amp -\frac{8}{3} \end{array} \right)\amp \amp\\ \frac{3}{7}R_2+R_3\to R_3\left( \begin{array}{rrr|r} 1 \amp \frac{2}{3} \amp \frac{1}{3} \amp \frac{1}{3}\\ 0 \amp 1 \amp 4 \amp -8\\ 0 \amp 0 \amp \frac{64}{21} \amp -\frac{128}{21} \end{array} \right)\amp \amp\\ \frac{21}{64}R_3\to R_3\left( \begin{array}{rrr|r} 1 \amp \frac{2}{3} \amp \frac{1}{3} \amp \frac{1}{3}\\ 0 \amp 1 \amp 4 \amp -8\\ 0 \amp 0 \amp 1 \amp -2 \end{array} \right)\amp \amp \end{align*}

Then we work from bottom back up:

\begin{align*} -4R_3 + R_2\to R_2\left( \begin{array}{rrr|r} 1 \amp \frac{2}{3} \amp \frac{1}{3} \amp \frac{1}{3}\\ 0 \amp 1 \amp 0 \amp 0\\ 0 \amp 0 \amp 1 \amp -2 \end{array} \right)\amp \amp\\ -\frac{1}{3}R_3 + R_1\to R_1\left( \begin{array}{rrr|r} 1 \amp \frac{2}{3} \amp 0 \amp 1\\ 0 \amp 1 \amp 0 \amp 0\\ 0 \amp 0 \amp 1 \amp -2 \end{array} \right)\amp \amp\\ -\frac{2}{3}R_2 + R_1\to R_1\left( \begin{array}{rrr|r} 1 \amp 0 \amp 0 \amp 1\\ 0 \amp 1 \amp 0 \amp 0\\ 0 \amp 0 \amp 1 \amp -2 \end{array} \right)\amp \amp \end{align*}

Which corresponds to the linear system:

\begin{align*} 1x+0y+0z\amp = 1\\ 0x+1y+0z\amp = 0\\ 0x+0y+z\amp = -2 \end{align*}

Which has solution \(x=1, y=0, z=2\text{.}\) To verify this works, we can check:

\begin{align*} 3(1)+2(0)+(-2)\amp = 1\\ (1)+(0)-(-2)\amp = 3\\ 2(1)-(0)+2(-2)\amp = -2 \end{align*}

and we have solved our linear system.

Subsection 2.2.2 Reduced Row Echelon Form using Technology.

Suppose we wanted to solve the system associated with the augmented matrix:

\begin{equation*} \left( \begin{array}{rrr|r} 2 \amp 3 \amp 1\amp 3\\ -2 \amp 4 \amp 2 \amp 0\\ 1 \amp -1 \amp 2 \amp 3 \end{array} \right)\text{.} \end{equation*}

We could solve this the same way we did Example 2.2.1, but your first thought here is probably "I don't want to do this." Yeah, me neither. One thing to note is that this process of finding a reduced row echelon form of a matrix is fairly mechanical. Thus, it is only natural that we should do this with a machine.

The first line defines the matrix \(A\) with the rows identical to the rows we defined in our above matrix. Then the code returns a printout of that matrix, and it's reduced row echelon form:

\begin{equation*} \left( \begin{array}{rrr|r} 1 \amp 0 \amp 0\amp 1\\ 0 \amp 1 \amp 0 \amp 0\\ 0 \amp 0 \amp 1 \amp 1 \end{array} \right) \end{equation*}

So \(x=1, y=0, z=1\text{.}\) Is this right? Well,

\begin{align*} 2(1)+3(0)+1\amp = 3\\ -2(1)+4(0)+2(1)\amp = 0\\ 1-0+2(1)\amp = 3 \end{align*}

Works for me.

By the way, if you enter the matrix from Example 2.2.1, and replace the first line with:

A=matrix(QQ, [[3,2,1,1],[1,1,1,3],[2,-1,2,-2]])

You should have the reduced row echelon matrix we achieved earlier.

Subsection 2.2.3 Infinitely many solutions

Consider the system:

\begin{align*} x+2y+z\amp = 3\\ -x+y+0z\amp = 0\\ 0x+3y+z\amp = 3 \end{align*}

Then by using rref using as \(A\text{:}\)

A=matrix(QQ, [[1,2,1,3],[-1,1,0,0],[0,3,1,3]])

We would obtain:

\begin{equation*} \left( \begin{array}{rrr|r} 1 \amp 0 \amp 1/3\amp 1\\ 0 \amp 1 \amp 1/3 \amp 1\\ 0 \amp 0 \amp 0 \amp 0 \end{array} \right) \end{equation*}

Which gives us the system:

\begin{align*} x+0y+(1/3)z\amp = 1\\ 0x+y+(1/3)z\amp = 1\\ 0x+0y+0z\amp = 0 \end{align*}

Well, the last line is useless, it basically says that \(0=0\text{,}\) which is true but useless. Let's break down the remaining equations:

\begin{align*} x+(1/3)z\amp = 1\\ y+(1/3)z\amp = 1 \end{align*}

Which can be thought of as:

\begin{align*} x\amp = 1-(1/3)z\\ y\amp = 1-(1/3)z \end{align*}

So basically what this is saying, no matter what \(z\) is, you can find \(x\) and \(y\text{.}\) So if \(z=0\text{,}\) then \(x,y=1\text{.}\) But if \(z=6\text{,}\) then \(x,y=-1\text{.}\) Well, what is \(z\text{?}\) Apparently it can be whatever we want, both of the above triplets provide perfectly legitimate solutions to the original system. What you see described is a line of solutions. The 3 equations describe 3 planes in 3 dimensional (\(x,y,z)\) space and they intersect at this line which can be describes as the solution to this equation.

Subsection 2.2.4 No Solutions

Consider a similar system:

\begin{align*} x+2y+z\amp = 3\\ -x+y+0z\amp = 0\\ 0x+3y+z\amp = 0 \end{align*}

Then by using rref using as \(A\text{:}\)

A=matrix(QQ, [[1,2,1,3],[-1,1,0,0],[0,3,1,0]])

We would obtain:

\begin{equation*} \left( \begin{array}{rrr|r} 1 \amp 0 \amp 1/3\amp 1\\ 0 \amp 1 \amp 1/3 \amp 1\\ 0 \amp 0 \amp 0 \amp 0 \end{array} \right) \end{equation*}

Which gives us the system:

\begin{align*} x+0y+(1/3)z\amp =\amp 1\\ 0x+y+(1/3)z\amp =\amp 1\\ 0x+0y+0z\amp =\amp 1 \end{align*}

Well, what does this last line mean? It says that \(0=1\text{.}\) When will this happen? Well basically never, no choice of \(x,y,z\) will ever make this true, our 3 planes do not intersect at a point or line or at all, so there is no solution.

Subsection 2.2.5 Applications

Example 2.2.2.

A new airline has recently purchased a fleet of Airbus A330-300s, Boeing 767-300ERs, and Boeing Dreamliner 787-9s to meet an estimated demand for 9,300 seats. The A330-300s seat 330 passengers and cost $250 million each, the 767-300ERs seat 270 passengers and cost $200 million each, while the 787-9s seat 240 passengers and cost $250 million each. The total cost of the fleet, which had twice as many 787-9s as 767s, was $8,100 million. How many of each type of aircraft did the company purchase?

Solution

It helps perhaps if we organize some of this information in a table.

\begin{equation*} \end{equation*}

By letting \(x\) denote the numbere of A330-300's, \(y\) denote the number of 767-300ERs and \(z\) denote the number of 787-9s, we obtain the following linear equalities:

First, the total capacity is 9,300 seats:

\begin{equation*} 330x+270y+240z=9300\text{.} \end{equation*}

Next, the total cost is $8,100 million:

\begin{equation*} 250x+200y+250z=8100\text{.} \end{equation*}

Finally, there are twice as many 787-9s as 767-300ERs:

\begin{align*} z\amp = 2y\\ -2y+z\amp = 0\text{.} \end{align*}

Giving us a system:

\begin{align*} 330x+270y+240z\amp = 9300\\ 250x+200y+250z\amp = 8100\\ -2y+z\amp = 0 \end{align*}

and corresponding augmented matrix:

\begin{equation*} \left( \begin{array}{rrr|r} 330 \amp 270 \amp 240\amp 9300\\ 250 \amp 200 \amp 250 \amp 8100\\ 0 \amp -2 \amp 1 \amp 0 \end{array} \right)\text{.} \end{equation*}

Using any method we'd like, we find row reduced echelon form:

\begin{equation*} \left( \begin{array}{rrr|r} 1 \amp 0 \amp 0\amp 10\\ 0 \amp 1 \amp 0 \amp 8\\ 0 \amp 0 \amp 1 \amp 16 \end{array} \right) \end{equation*}

which corresponds to \(x=10, y=8, z=16\) or 10 A330-300s, 8 767300ERs and 16 787-9s.

To verify that this is a legitimate solution, check:

\begin{align*} 330(10)+270(8)+240(16)\amp = 9300\\ 250(10)+200(8)+250(16)\amp = 8100\\ -2(8)+16\amp = 0\text{.} \end{align*}
Example 2.2.3.

Margo needs 200mg of vitamin A, 100mg of vitamin D, and 140mg of vitamin E per week. She has three supplements: the first contains 20% vitamin A, 20% vitamin D and 20% vitamin E; the second contains 10% vitamin A, 30% vitamin D and 40% vitamin E; the third contains 50% vitamin A, 10% vitamin D and 20% vitamin E. How much of each supplement should she eat each week?

Solution

Let \(x,y,z\) denote the quantities of supplements 1,2 and 3 she takes each week. She needs 200mg of vitamin A, which gives us:

\begin{equation*} 0.2x+0.1y+0.5z=200\text{.} \end{equation*}

Similarly, for 100mg of vitamin D:

\begin{equation*} 0.2x+0.3y+0.1z=100 \end{equation*}

and for 140mg of vitamin E:

\begin{equation*} 0.2x+0.4y+0.2z=140\text{.} \end{equation*}

This gives us the system:

\begin{align*} 0.2x+0.1y+0.5z\amp = 200\\ 0.2x+0.3y+0.1z\amp = 100\\ 0.2x+0.4y+0.2z\amp = 140 \end{align*}

with corresponding matrix:

\begin{equation*} \left( \begin{array}{rrr|r} 0.2 \amp 0.1 \amp 0.5\amp 200\\ 0.2 \amp 0.3 \amp 0.1 \amp 100\\ 0.2 \amp 0.4 \amp 0.2 \amp 140 \end{array} \right)\text{.} \end{equation*}

This has reduced row echelon form:

\begin{equation*} \left( \begin{array}{rrr|r} 1 \amp 0 \amp 0\amp 200\\ 0 \amp 1 \amp 0 \amp 100\\ 0 \amp 0 \amp 1 \amp 300 \end{array} \right) \end{equation*}

which corresponds to solution \(x=200, y=100, z=300\) or 200 mg of supplement 1, 100mg of supplement 2 and 300mg of supplement 3.

Example 2.2.4.

Suppose an event with 300 people ends, and the attendees walk from the venue to two restaurants, Xavier's and Yoneda's. They walk along streets Anderson, Birmingham and Central, which are all one way. When the night ends, there are 75 people at Xavier's and 225 at Yoneda's.

Let \(a,b,c\) denote the number of people walking along Anderson, Birmingham and Central.

  1. Find an expression for the amount of people walking along each street.
  2. If 50 people crossed central, how many people crossed Anderson and Birmingham?
  3. If 100 people go through Anderson, how many go through Birmingham or Central?
Solution 1

We first set up our systems of equations. We first note that all 300 people have to walk either along either Anderson or Birmingham, so

\begin{equation*} a+b=300\text{.} \end{equation*}

Next, the 75 at Xavier is the however many people walked down Anderson, and take away whoever veered off onto Central so:

\begin{equation*} a-c=75\text{.} \end{equation*}

The 225 people at Yoneda is everyone coming across Birmingham and Central so

\begin{equation*} b+c=225\text{.} \end{equation*}

Thus the system is:

\begin{align*} a+b\amp = 300\\ a-c\amp = 75\\ b+c\amp = 225\text{,} \end{align*}

with associated augmented matrix:

\begin{equation*} \left( \begin{array}{rrr|r} 1 \amp 1 \amp 0\amp 300\\ 1 \amp 0 \amp -1 \amp 75\\ 0 \amp 1 \amp 1 \amp 225 \end{array} \right)\text{.} \end{equation*}

So the reduced row echelon form of this is:

\begin{equation*} \left( \begin{array}{rrr|r} 1 \amp 0 \amp -1 \amp 75\\ 0 \amp 1 \amp 1 \amp 225\\ 0 \amp 0 \amp 0\amp 0 \end{array} \right)\text{.} \end{equation*}

What this tells us is that there is not a unique solution, as the system associated with this matrix is:

\begin{align*} a-c\amp = 75\\ b+c\amp = 225\text{,} \end{align*}

Which is two of the original equations. So, depending on what \(c\) is, this will determine how many people traverse Anderson and Birmingham.

Solution 2

If 50 people crossed central, how many people crossed Anderson and Birmingham? So letting \(c=50\text{,}\) we would have:

\begin{align*} a-c\amp = 75\\ a-50\amp = 75\\ a\amp = 125\\ b+c\amp = 225\\ b+50\amp = 225\\ b\amp = 175\text{,} \end{align*}

So 125 people across Anderson and 175 across Birmingham. Another way to do this would be to add an equation \(c=50\) to our system:

\begin{equation*} \left( \begin{array}{rrr|r} 1 \amp 1 \amp 0\amp 300\\ 1 \amp 0 \amp -1 \amp 75\\ 0 \amp 1 \amp 1 \amp 225\\ 0 \amp 0 \amp 1 \amp 50 \end{array} \right)\text{.} \end{equation*}

Which has reduced row echelon form:

\begin{equation*} \left( \begin{array}{rrr|r} 1 \amp 0 \amp 0 \amp 125\\ 0 \amp 1 \amp 0 \amp 175\\ 0 \amp 0 \amp 1\amp 50\\ 0 \amp 0 \amp 0\amp 0 \end{array} \right)\text{.} \end{equation*}

That is \(a=125, b=125, c=50\) which is our solution above.

Solution 3

If 100 people go through Anderson, how many go through Birmingham or Central? We can revisit our linear system from (a):

\begin{align*} a-c\amp = 75\\ 100-c\amp = 75\\ c\amp = 25\\ b+c\amp = 225\\ b+25\amp = 225\\ b\amp = 200\text{,} \end{align*}

so 200 across Birmingham and 25 across Central. We can also add the linear equation \(a=100\text{:}\)

\begin{equation*} \left( \begin{array}{rrr|r} 1 \amp 1 \amp 0\amp 300\\ 1 \amp 0 \amp -1 \amp 75\\ 0 \amp 1 \amp 1 \amp 225\\ 1 \amp 0 \amp 0 \amp 100 \end{array} \right) \end{equation*}

with reduced row echelon form:

\begin{equation*} \left( \begin{array}{rrr|r} 1 \amp 0 \amp 0 \amp 100\\ 0 \amp 1 \amp 0 \amp 200\\ 0 \amp 0 \amp 1\amp 25\\ 0 \amp 0 \amp 0\amp 0 \end{array} \right)\text{.} \end{equation*}

So 100 across Anderson, 200 across Birmingham and 25 across Central.

Checkpoint 2.2.5.

An Ice Cream Manufacturer makes three flavors, Avacado, Blueberry and Chocolate. A batch of Avacdo requires 2 eggs, 1 cup of milk and 2 cups of cream. A batch of blueberry Blueberry requires 1 egg, 1 cup of milk and 2 cups of cream. A batch of Chocolate requires 1 egg, 2 cups of milk and 1 cup of cream. The manufacturer has 350 eggs, 350 cups of milk, and 400 cups of cream. How many batches of each flavor should be made to use up all of the ingredients?

Solution

Let us suppose that \(a\) represents the number of Avacado batches, \(b\) blueberry and \(c\) chocolate (wildly creative of us).

Then the amount of eggs used is: 2 per avacado batch, 1 per blueberry and 1 per chocolate. If we use 350 eggs, this gives us:

\begin{equation*} 2a+b+c=350. \end{equation*}

The amount of milk used is: 1 cup per avacado batch, 1 per blueberry and 2 per chocolate. If we use 350 cups of milk, this gives us:

\begin{equation*} a+b+2c=350. \end{equation*}

The amount of cream used is: 2 cups per avacado batch, 2 per blueberry and 1 per chocolate. If we use 400 cups of cream, this gives us:

\begin{equation*} 2a+2b+c=350. \end{equation*}

Putting all this together gives us:

\begin{align*} 2a+b+c \amp=350\\ a+b+2c \amp=350\\ 2a+2b+c \amp=400 \end{align*}

Which has matrix form:

\begin{equation*} \left( \begin{array}{rrr|r} 2 \amp 1 \amp 1 \amp 350\\ 1 \amp 1 \amp 2 \amp 350\\ 2 \amp 2 \amp 1 \amp 400 \end{array} \right) \end{equation*}

Which in turn has reduced row echelon form:

\begin{equation*} \left( \begin{array}{rrr|r} 1 \amp 0 \amp 0 \amp 100\\ 0 \amp 1 \amp 0 \amp 50\\ 0 \amp 0 \amp 1 \amp 100 \end{array} \right) \end{equation*}

So 100 batches of Avacado, 50 batches of Blueberry and 100 batches of Chocolate does the trick!

Checkpoint 2.2.6.

An investor purchases stocks in three different companies. Overpriced Textbooks Inc., Sandwich Solutions and Union Holdings Inc.. Their prices and dividend yields are as follows:

\begin{equation*} \begin{array}{|c|c|c|} \hline \text{Company} \amp \text{Price} \amp \text{Dividend Yield}\\ \hline \text{Overpriced Textbooks Inc.} \amp \$56 \amp 2\%\\ \text{Sandwich Solutions} \amp \$16 \amp 7\%\\ \text{Union Holdings Inc.} \amp \$80 \amp 2\%\\ \hline \end{array} \end{equation*}

She invested a total of $8,400 in shares of the three stocks at the given prices and expected to earn $248 in annual dividends. If she purchased a total of 200 shares, how many shares of each stock did she purchase?

Solution

Let \(o, s, u\) represent the number of shares in Overpriced Textbooks Inc., Sandwich Solutions and Union Holdings Inc. respectively.

Notice that she pays $56 per share of Overpriced Textbooks Inc., $16 per share of Sandwich Solutions and $80 per share of Union Holdings Inc.. So if this totals $8400, then:

\begin{equation*} 56o+16s+80u=8400. \end{equation*}

Then note: each share of Overpriced Textbooks Inc. pays \(56\cdot 0.02=1.12\) in dividends, a share of Sandwich Solutions pays \(16\cdot 0.07=1.12\) in dividends and each share of Union Holdings Inc. pays \(80\cdot 0.02=1.60\) in dividends. So if this totals $248, then:

\begin{equation*} 1.12o+1.12s+1.6u=248. \end{equation*}

Finally,she bought 200 shares, so

\begin{equation*} o+s+u=200. \end{equation*}

Putting all this together gives us:

\begin{align*} 56o+16s+80u\amp=8400\\ 1.12o+1.12s+1.6u\amp=248\\ o+s+u\amp=200 \end{align*}

Which has matrix form:

\begin{equation*} \left( \begin{array}{rrr|r} 56 \amp 16 \amp 80 \amp 8400\\ 1.12 \amp 1.12 \amp 1.6 \amp 248\\ 1 \amp 1 \amp 1 \amp 200 \end{array} \right) \end{equation*}

Which in turn has reduced row echelon form:

\begin{equation*} \left( \begin{array}{rrr|r} 1 \amp 0 \amp 0 \amp 50\\ 0 \amp 1 \amp 0 \amp 100\\ 0 \amp 0 \amp 1 \amp 50 \end{array} \right) \end{equation*}

So 50 shares of Overpriced Textbooks Inc., 100 shares of Sandwich Solutions and 50 shares of Union Holdings Inc.

Checkpoint 2.2.7.

A company is using trucks vans and SUV's to ship prototype robots. There are three models: A, B and C. A truck holds 2 model A's, 2 model B's and 1 model C. A van holds 3 A's, 2B's, 3 C's. A SUV holds 5A's, 6B's and 2C's.

  1. How many trucks, vans and SUV's would you need to ship 42 model A's, 40 model B's, and 26 model C's.
  2. How many of each type of vehicles could you use if model C's were discontinued?
Solution 1

Let's let \(t\) be the number of trucks, \(v\) the number of vans and \(s\) be the number of SUV's.

Each truck holds 2 model A's, each van 3, and each SUV 3. Since 28 model A's are shipped, this is results in

\begin{equation*} 2t+3v+5s=42. \end{equation*}

Each truck holds 2 model B's, each van 2, and each SUV 6. Since 30 model B's are shipped, this is results in

\begin{equation*} 2t+2v+6s=40. \end{equation*}
x

Each truck holds 1 model C, each van 3, and each SUV 2. Since 16 model C's are shipped, this is results in

\begin{equation*} t+3v+2s=26. \end{equation*}

Putting all this together gives us:

\begin{align*} 2t+3v+5s\amp =42\\ 2t+2v+6s \amp =40\\ t+3v+2s \amp=26 \end{align*}

Which has matrix form:

\begin{equation*} \left( \begin{array}{rrr|r} 2 \amp 3 \amp 5 \amp 42\\ 2 \amp 2 \amp 6 \amp 40\\ 1 \amp 3 \amp 2 \amp 26 \end{array} \right) \end{equation*}

Which in turn has reduced row echelon form:

\begin{equation*} \left( \begin{array}{rrr|r} 1 \amp 0 \amp 0 \amp 10\\ 0 \amp 1 \amp 0 \amp 4\\ 0 \amp 0 \amp 1 \amp 2 \end{array} \right) \end{equation*}

So 10 trucks, 4 vans and 2 SUV's.

Solution 2

If we remove the model C requirement, we have the system:

\begin{align*} 2t+3v+5s\amp =42\\ 2t+2v+6s \amp =40 \end{align*}

Which in turn has reduced row echelon form:

\begin{equation*} \left( \begin{array}{rrr|r} 1 \amp 0 \amp 4 \amp 18\\ 0 \amp 1 \amp -1 \amp 2\\ \end{array} \right) \end{equation*}

In other words:

\begin{align*} t+4s\amp =18\\ v-s\amp =2 \end{align*}

So there is more than 1 solution. Remember that the number of any type of vehicle has to be a non-negative whole number.

  1. So if you had \(s=0\) SUV's, then you must have \(t=18-4\cdot 0=18\) trucks and \(v=2+0=2\) vans.
  2. If you had \(s=1\) SUV's, then you must have \(t=18-4\cdot 1=14\) trucks and \(v=2+1=3\) vans.
  3. If you had \(s=2\) SUV's, then you must have \(t=18-4\cdot 2=10\) trucks and \(v=2+2=4\) vans.
  4. If you had \(s=3\) SUV's, then you must have \(t=18-4\cdot 3=6\) trucks and \(v=2+3=5\) vans.
  5. If you had \(s=4\) SUV's, then you must have \(t=18-4\cdot 4=2\) trucks and \(v=2+4=6\) vans.
  6. If you had \(s=5\) SUV's, then you must have \(t=18-4\cdot 5=-2\) trucks which is impossible.

So the valid number of truck, van and SUV's are: \((18,2,0), (14,3,1), (10,4,2), (6,5,3), (2,6,4).\)
