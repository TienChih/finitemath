Section 7.7 Derivative of Logarithms

We recall that log functions are inverses of exponential functions. We first conceptualized them in Section 6.6 as reflections of exponentials across the \(y=x\) line. It is sensible then that the derivatives of logs should be based on those of exponentials.

In fact, we can see it below. Each point on the \(y=e^x\) line corresponds to a point on the \(y=\ln(x)\) line, and the slopes are reciprocals of each other:

Subsection 7.7.1 Computing Logarithmic Derivatives

We can use the chain rule to find the derivative of log functions as well. Let's first compute \(\frac{d}{dx}[\ln(x)]\text{:}\)

\begin{align*} x\amp = e^{\ln(x)}\\ \frac{d}{dx}[x]\amp = \frac{d}{dx}[e^{\ln(x)}]\\ 1\amp = e^{\ln(x)}\frac{d}{dx}[\ln(x)]\\ 1\amp = x\frac{d}{dx}[\ln(x)]\\ \frac{d}{dx}[\ln(x)]\amp = \frac{1}{x}\text{.} \end{align*}

Visually:

For general \(\log_a(x)\text{:}\)

\begin{align*} x\amp = a^{\log_a(x)}\\ \frac{d}{dx}[x]\amp = \frac{d}{dx}[a^{\log_a(x)}]\\ 1\amp = a^{\log_a(x)}\ln(a)\frac{d}{dx}[\log_a(x)]\\ 1\amp = x\ln(a)\frac{d}{dx}[\log_a(x)]\\ \frac{d}{dx}[\log_a(x)]\amp = \frac{1}{x\ln(a)}\text{.} \end{align*}

Subsection 7.7.2 Exercises

Example 7.7.1.

Let \(g(x)=\ln(x^2+1)\text{.}\) Find \(g'(x)\text{.}\)

Solution

Notice:

\begin{align*} g'(x)\amp = \frac{d}{dx}[\ln(x^2+1)]\\ \amp =_{CR} \frac{1}{x^2+1}\frac{d}{dx}[x^2+1]\\ \amp = \frac{2x}{x^2+1}\text{.} \end{align*}

Checkpoint 7.7.2.

Suppose the demand function for q units of a certain item is

\begin{equation*} p=D(q)=150+\frac{40}{\ln(q)}, q>1, \end{equation*}

where \(p\) is in dollars.

  • Find the marginal revenue.
  • Approximate the revenue from one more unit when 6 units are sold.
Solution

Notice:

  • We note that revenue is \(R(q)=qD(q)=150q+\frac{40q}{\ln(q)}\text{.}\) We then differentiate \(R(q)\text{:}\)
    \begin{align*} D'(q)\amp = \frac{d}{dq}[150q+\frac{40q}{\ln(q)}]\\ \amp = 150 +\frac{d}{dq}[\frac{40q}{\ln(q)}]\\ \amp =_{QR} 150 +\frac{\frac{d}{dq}[40q]\ln(q)-40q\frac{d}{dq}[\ln(q)]}{(\ln(q))^2}\\ \amp = 150 +\frac{40\ln(q)-40q\frac{1}{q}}{(\ln(q))^2}\\ \amp = 150 +\frac{40\ln(q)-40}{(\ln(q))^2}\text{.} \end{align*}
  • This would be \(R'(6)= 150 +\frac{40\ln(6)-40}{(\ln(6))^2}\approx 159.87\) or \(\$159.87\) dollars per unit sold.
