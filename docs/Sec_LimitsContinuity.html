Section 7.1 Limits & Continuity

In this section, we discuss the idea of limits, or the math about approaching something infinitesimally closely.
Definition 7.1.1.

Informally, the limit of a function \(\limit{x}{a} f(x)=L\) (if it exists) is a number \(L\) such that when \(f(x)\) get's closer and closer to \(a\text{,}\) \(f(x)\) gets closer and closer to \(L\text{.}\)

Subsection 7.1.1 The Limit

Consider the function \(f(x)=\frac{x^2-1}{x-1}\text{,}\) what happens as \(x\) get's closer and closer to 1?

\begin{equation*} \begin{array}{|c|c|} \hline x \amp f(x)\\ \hline 1.1 \amp 2.1\\ 1.01 \amp 2.01\\ 1.001 \amp 2.001\\ .9 \amp 1.9\\ .99\amp 1.99\\ .999 \amp 1.999\\ \hline \end{array} \end{equation*}

You can test and see these values here: It certainly seems as if \(x\) gets closer and closer to \(1\text{,}\) \(f(x)\) approaches the value \(2\text{.}\) Why couldn't we just plug in 1? Because if we did we would get \(f(1)"="\frac{1^2-1}{1-1}"="\frac{0}{0}\) which is undefined. Recall that \(1\) would not be a part of the domain of \(f(x)\text{.}\) This, in fact, is the whole point of limits, the entire reason they exist, is in order to give values to things like \(f(1)\) which are not defined.

The left and right limits of a function are values the function approaches as \(x\) \(a\) from the left or the right, so as we see above, they both (the black and orange dots in the linked graph) both approach the same value.

Definition 7.1.2.

We use \(\limit{x}{a^-} f(x)\) to denote the limit of \(f(x)\) as \(x\) approaches \(a\) from the left. Similarly \(\limit{x}{a^+} f(x)\) denotes the limit of \(f(x)\) as \(x\) approaches \(a\) from the right. (If they exist).

So:

  • \(\limit{x}{1^-}\frac{x^2-1}{x-1}=2\text{.}\)

  • \(\displaystyle \limit{x}{1^+}\frac{x^2-1}{x-1}=2\)

  • \(\displaystyle \limit{x}{1}\frac{x^2-1}{x-1}=2\)

  • \(f(1)\) is undefined.

Example 7.1.3. Limits of a function.

So looking at this graph of \(y=f(x)\text{:}\)

For \(x=2,4,6,8\text{,}\) find the left and right limits, the limit, and the value of the function. See if you understand why the following are true:

  • \(\limit{x}{2^-}f(x)=2\text{,}\) as we approach \(x\) from the left, this is what \(f(x)\) approaches.

  • \(\limit{x}{2^+}f(x)=1\text{,}\) as we approach \(x\) from the right, this is what \(f(x)\) approaches.

  • \(\limit{x}{2}f(x)\) is undefined, there is no single number that \(f(x)\) get's closer to, so there is no limit.

  • \(f(2)=1\text{,}\) that's the actual value of \(f(2)\text{.}\)

Similarly, we should see:

  • \(\limit{x}{4^-}f(x)=1\text{.}\)

  • \(\limit{x}{4^+}f(x)=1\text{.}\)

  • \(\limit{x}{4}f(x)=1\text{.}\)

  • \(f(4)=-1\text{.}\)

  • \(\limit{x}{6^-}f(x)=-1\text{.}\)

  • \(\limit{x}{6^+}f(x)=-1\text{.}\)

  • \(\limit{x}{6}f(x)=-1\text{.}\)

  • \(f(6)=-1\text{.}\)

  • \(\limit{x}{8^-}f(x)=-3\text{.}\)

  • \(\limit{x}{8^+}f(x)=-3\text{.}\)

  • \(\limit{x}{8}f(x)=-3\text{.}\)

  • \(f(8)\) is undefined.

A great shorthand for determining when there is or isn't a limit is: \(\limit{x}{a}f(x)=L\) and exists if and only if \(\limit{x}{a^-}f(x)=L\) and \(\limit{x}{a^+}f(x)=L.\)

Subsection 7.1.2 Continuity

Most people have an intuitive understanding of what it means for a function to be continuous, that it doesn't have any jumps, and the curve of the function “connect”. This leads to a bit more formal of a definition:

Definition 7.1.5.

A function \(f(x)\) is continuous at \(x=a\) if \(\limit{x}{a}f(x)=f(a).\) It is continuous if it is continuous at each \(x\) in it's domain.

Example 7.1.6. Points of continuity and discontinuity.

In Example 7.1.3, \(f(x)\) is continuous at each \(x\) except for \(x=2, 4\text{,}\) since the limits are either undefined or don't agree with the function at these values.

Checkpoint 7.1.7. Identify points of continuity.

In Checkpoint 7.1.4, identify when \(g(x)\) is continuous.

Solution

We can see that \(g(x)\) is continuous everywhere but at \(x=0,4\text{.}\) We note that \(g(x)\) has no defined limit at \(x=0\text{,}\) and \(g(x)\) is undefined when \(x=4\text{.}\)

Example 7.1.8. Identifying points of continuity.

For what values of \(x\) is

\begin{equation*} f(x)=\begin{cases} 2^x \amp x\lt 0 \\ x+1 \amp 0\leq x \leq 4 \\ \sqrt{x} \amp x>4\end{cases} \end{equation*}

continuous?

Solution

We note that we only really need to check \(x=0\) and \(x=4\text{.}\) At \(x=0\text{,}\) we have

\begin{equation*} \limit{x}{0^-} f(x)=2^0=1, \limit{x}{0^+} f(x)=0+1=1.\text{.} \end{equation*}

Thus \(\limit{x}{0}f(x)=1\text{.}\) Since \(f(0)=0+1=1\) we have that \(f(x)\) is continuous at \(x=0\text{.}\)

On the other hand, at \(x=4\text{,}\) we have

\begin{equation*} \limit{x}{4^-} f(x)=4+1=5, \limit{x}{4^+} f(x)=\sqrt{4}=2.\text{.} \end{equation*}

Thus \(\limit{x}{4}f(x)\) is undefined and \(f(x)\) is not continuous at \(x=4\text{.}\)

Example 7.1.9. Ensuring continuity of a function.

For what value of \(k\) is

\begin{equation*} g(x)=\begin{cases} -x^2+4 \amp x\leq 1 \\ kc \amp x>1\end{cases} \end{equation*}

continuous?

Solution

We note that \(\limit{x}{1^-}g(x)=g(1)=-1+4=3\text{.}\) We also note that \(\limit{x}{1^+}g(x)=k(1)=k\text{.}\) So for \(g(x)\) to be continuous at \(x=1\text{,}\) we need

\begin{equation*} 3=\limit{x}{1^+}g(x)=k. \end{equation*}

Thus \(k=3\text{.}\)

We observe this visually: Slide the value of \(k\) to see how other values of \(k\) will not give you a continuous function.

Checkpoint 7.1.10. Ensuring continuity 1.

For what values of \(x\) is

\begin{equation*} a(x)=\begin{cases} x \amp x\lt 1 \\ x^2+1 \amp 1\leq x \lt 3 \\ 2^x \amp x\geq 3\end{cases} \end{equation*}

continuous?

Solution

We note that we only really need to check \(x=1\) and \(x=3\text{.}\) At \(x=1\text{,}\) we have

\begin{equation*} \limit{x}{1^-} a(x)=1, \limit{x}{1^+} a(x)=1^2+1=2.\text{.} \end{equation*}

Thus \(\limit{x}{1}a(x)\) is undefined and \(a(x)\) is not continuous at \(x=1\text{.}\)

At \(x=3\text{,}\) we have

\begin{equation*} \limit{x}{3^-} a(x)=3^2+1=10, \limit{x}{3^+} a(x)=2^3=8. \end{equation*}

Thus \(\limit{x}{3}\) is undefined and \(f(x)\) is not continuous at \(x=3\text{.}\)

Checkpoint 7.1.11. Ensuring continuity 2.

For what value of \(k\) is

\begin{equation*} b(x)=\begin{cases} 2^x \amp x\lt 2 \\ -x+k \amp x\geq2\end{cases} \end{equation*}

continuous?

Solution

We note that \(\limit{x}{2^-}b(x)=2^2=4\text{.}\) We also note that \(\limit{x}{2^+}b(x)=b(2)=-(2)+k=-2+k\text{.}\) So for \(b(x)\) to be continuous at \(x=2\text{,}\) we need

\begin{equation*} 4=\limit{x}{2^+}b(x)=b(2)=-2+k. \end{equation*}

Thus \(k=6\text{.}\)

We observe this visually: Slide the value of \(k\) to see how other values of \(k\) will not give you a continuous function.
