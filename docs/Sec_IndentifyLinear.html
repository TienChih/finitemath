Section 1.3 Identifying a Linear Function from information.

It is often the case that when one works with a linear function, one will not be given the exact form for the function. You may for example only know the rate of change, or the value of the function at a few points. To be able to make the most of this situation, it would be good to be able to still identify the linear function in question, when possible.

What is the minimal amount of information necessary to identify a linear function?

Here is where our geometric interpretation of these functions is helpful. If someone handed you a sheet of paper and asked you to draw a line, how much information would they have to give you so there could be only one like you'd be able to draw?

Is a point enough?

If someone drew a dot on a piece of paper, there is certainly a line you could draw through it, but it definitely does seem like there's infinitely many lines you could draw as well. Take \((2,3)\text{:}\)

If someone gave you a piece of paper and told you would direction the line should take, there is definitely a line you could draw with that direction, but depending where you start, there's infinitely many lines you could draw as well. Take \(m=\frac{3}{2}\text{:}\)

So this begs the question, what IS the minimum amount of information that you'd need to uniquely identify a line?

Subsection 1.3.1 Point & Slope

If we combined the two pieces of information from the start of the section, would we obtain a unique line? Intuitively this should make sense, there are infinitely many lines which go through a point, but only one with slope \(m=\frac{3}{2}\text{,}\) similarly, there are infinitely many lines with slope \(\frac{3}{2}\text{,}\) but they all pass through different points, only one of them passes through \((2,3)\text{.}\) Again to refer to our geometric ideas, if someone drew a point on a paper, and told you which direction you had to go from there, there's really only one line you could draw.

How does this translate Algebraically?

So given a linear function is defined by the slope \(m\) and intercept \(b\text{.}\) By the discussion above, if we are given a slope \(\color{red}{m}\) and a pair of points \(\color{blue}{(x_0,y_0)}\text{,}\) we should be able to uniquely identify \(b\text{.}\)

Example 1.3.2.

Find the linear function with slope 3 passing through \((2,5)\text{.}\)

Note that we are given \(\color{red}{m=3}\) and \(\color{blue}{(x_0,y_0)=(2,5)}\text{.}\) The general form for a line is \(y=mx+b\text{,}\) thus:

\begin{align*} y\amp = mx+b\\ \color{blue}{5}\amp = \color{red}{3}\cdot\color{blue}{2}+b\\ 5\amp = 6+b\\ b\amp = 5-6=-1\text{.} \end{align*}

Thus \(y=3x-1\) is the linear equation in question, \(m=3, b=-1\text{.}\)

We can verify that this line has slope \(m=3\) and when \(x=2, y=3\cdot 2-1=6-1=5\text{.}\)

Subsection 1.3.2 Point-Slope form.

So, typically lines are defined in the form \(y=mx+b\text{,}\) where \(m\) is the slope and \(b\) is the \(y\)-intercept. It can be convenient to also define a line in the form

\begin{equation*} y-y_0=m(x-x_0)\text{,} \end{equation*}

where \((x_0, y_0)\) is a point that falls on your line.

Why would we care about a second formulation, and how do we know it's even a line? To answer the first question, we recall that ANY geometric object defined by an equation is the set of points that make the equation true. So given the equation \(y-y_0=m(x-x_0)\text{,}\) if we plug in \((x_0, y_0)\text{,}\) we would get \(y_0-y_0=m(x_0-x_0)\) or \(0=0\) which is definitely a true statement. So whatever shape we get from \(y-y_0=m(x-x_0)\text{,}\) it contains \((x_0, y_0)\text{.}\)

We know it's a line because:

\begin{align*} y-y_0\amp = m(x-x_0)\\ y\amp = mx-mx_0+y_0\\ y\amp = mx+(y_0-mx_0) \end{align*}

so if we let \(b=y_0-mx_0\text{,}\) we get the standard form for a line.

Example 1.3.3.

Find the line with slope 3 passing through \((2,5)\text{,}\) again.

Using the point-slope form, knowing that \(\color{blue}{(2,5)}\) lies on the line, and that \(\color{red}{m}\text{:}\)

\begin{align*} y-y_0\amp = m(x-x_0)\\ y-\color{blue}{5}\amp = \color{red}{3}(x-\color{blue}{2})\\ y\amp = 3x-6+5\\ y\amp = 3x-1\text{.} \end{align*}

Either way, you obtain the same line, with slope 3, passing through (2,5).

Subsection 1.3.3 Two Points

On the other hand, instead of giving you a point and a direction, one could be given two points to traverse through. If you imagine any two dots on a piece of paper, one can imagine that you may draw a line between them. The exact way we do this also follows intuitively. If you were at home, and you had to go to the store, the natural steps you would take to do so are:

  1. Figure out which way to the store.

  2. Go there.

So we have to figure out the direction between these points, which in our analogy means the slope between the points. Remember the slope measures how much \(y\) changes per change in \(x\text{,}\) or \(m=\frac{\Delta y}{\Delta x}\text{.}\) So, the measurement of slope follows from the change in \(y\) over the change in \(x\text{.}\) So to measure this, if we have points \((x_0, y_0), (x_1, y_1)\text{,}\) we want to see how much \(y\) changes over how much \(x\) changes, or:

\begin{equation*} m=\frac{\Delta y}{\Delta x}=\frac{y_1-y_0}{x_1-x_0} \end{equation*}

or equivalently \(m=\frac{y_0-y_1}{x_0-x_1}\text{.}\)

Example 1.3.4.

What line passes between \((2,3)\) and \((4,1)\text{?}\)

So we note then that \(\color{red}{m=\frac{1-3}{4-2}=-1}\) (also \(\color{red}{m=\frac{3-1}{2-4}=-1}\)). Once the slope is identified, we can use any point, and any method to find the line. Once we pick a point, We have a problem that follows the form of Examples 1.3.2, Example 1.3.3.

We won't go through all the possibilities but if we considered the point \(\color{blue}{(2,3)}\text{:}\)

\begin{align*} y\amp = mx+b\\ \color{blue}{3}\amp = \color{red}{-1}\cdot \color{blue}{2}+b\\ b\amp = 3+2=5\\ y\amp = -x+5\text{.} \end{align*}

Or if we picked \(\color{blue}{(1,4)}\text{:}\)

\begin{align*} y-y_0\amp = m(x-x_0)\\ y-\color{blue}{1}\amp = \color{red}{-1}\cdot(x-\color{blue}{4})\\ y\amp = -x+4+1\\ y\amp = -x+5\text{.} \end{align*}

Either way, we identify the same line:

Subsection 1.3.4 Utilizing the Linear Function

Once you obtain a linear function, this gives you a relationship between the \(x\) and \(y\) variables. It allows you to identify, given an \(x\) or \(y\) value, the other value:

Example 1.3.5.

Consider the linear function \(y=2x-3\text{.}\)

  1. When \(y=0\text{,}\) what is \(x\text{?}\)

  2. When \(x=3\) what is \(y\text{?}\)

These can be identified algebraically:

  1. When \(\color{ForestGreen}{y=0}\text{,}\) we have:

    \begin{align*} y\amp =\amp 2x-3\\ \color{ForestGreen}{0}\amp =\amp 2x-3\\ 3\amp =\amp 2x\\ x\amp =\amp 1.5\text{.} \end{align*}

  2. If \(\color{orange}{x=3}\text{:}\)

    \begin{align*} y\amp = 2x-3\\ y\amp = 2(\color{orange}{3})-3\\ y\amp = 6-3=3\text{.} \end{align*}
