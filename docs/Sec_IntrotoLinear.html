Section 1.1 What is a Linear Function?

A linear function is a function whose inputs and outputs are real numbers, and the change in output per unit change increase in input is always the same. A linear function may always increase, or decrease, or stay the same, but the rate at which it does so will never change.

Example 1.1.1.

Consider a function who takes on the following values:

\begin{equation*} \begin{array}{|c||c|c|c|c|c|} \hline x \amp 1\amp 2\amp 3\amp 4\amp 10\\ \hline f(x)\amp 10\amp 12\amp 14\amp 16\amp 28\\ \hline \end{array} \end{equation*}

Does this function appear linear?

Solution

It certainly takes on the semblance of linearity, if we observe the change from \(x=1\) to \(x=2\text{,}\) the change in \(f(x)\) or \(\Delta f(x)=12-10=2\text{.}\)

When we go from \(x=2\) to \(x=3\text{,}\) we again saw that \(\Delta f(x)=14-12=2\text{,}\) and so forth.

Even when we go from \(x=4\) to \(x=10\text{,}\) we say that \(\Delta f(x)=28-16=12\text{,}\) but that was over a change of \(\Delta x=10-4=6\text{.}\) This is still a change of \(12/6=2\) per unit of \(x\text{.}\) So assuming nothing wild happens between the listed values, this function does seem linear.

Example 1.1.4.

Consider a function who takes on the following values:

\begin{equation*} \begin{array}{|c||c|c|c|c|c|} \hline x \amp -2\amp -1\amp 0\amp 1\amp 2\\ \hline f(x)\amp 4\amp 1\amp 0\amp 1\amp 4\\ \hline \end{array} \end{equation*}

Does this function appear linear?

Solution

If we observe the change from \(x=-2\) to \(x=-1\text{,}\) the change in \(f(x)\) or \(\Delta f(x)=1-4=-3\text{.}\)

However, when we go from \(x=-1\) to \(x=0\text{,}\) we see that \(\Delta f(x)=0-1=-1\text{.}\) Similarly, when we go from \(x=0\) to \(x=1\text{,}\) we see that \(\Delta f(x)=1-0=1\text{,}\) and when we go from \(x=1\) to \(x=2\text{,}\) we see that \(\Delta f(x)=4-1=3\text{.}\)

Thus, this function goes from decreasing, to increasing, and the rate at which it does so changes, thus this function is not a linear function.
