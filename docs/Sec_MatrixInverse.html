Section 2.4 Matrix Inverses

In Definition 2.3.5, we saw the introduction of an identity matrix. This matrix plays the role of multiplying by 1. So, given another matrix \(A\text{,}\) is there a matrix \(B\) so that \(AB=I_n\text{?}\)

Subsection 2.4.1 Direct Inversion

We begin with a motivating example: Consider matrices

\begin{equation*} A=\begin{pmatrix} 4 \amp 1 \\ 3 \amp 1\end{pmatrix}, B=\begin{pmatrix} 1 \amp -1 \\ -3 \amp 4\end{pmatrix}. \end{equation*}

When we take their products, we notice:

Thus \(AB=I_2=BA\text{.}\) Thus \(A\) and \(B\) are inverses of each other.

Definition 2.4.1.

Given an \(n\times n\) matrix \(A\text{,}\) it's inverse is an \(n\times n\) matrix denoted \(A^{-1}\) such that

\begin{equation*} AA^{-1}=I_n=A^{-1}A. \end{equation*}

So lets say we wanted to find the inverse of

\begin{equation*} M= \begin{pmatrix} 3 \amp 1 \\ 5 \amp 2\end{pmatrix} \end{equation*}

we want a matrix \(M^{-1}=\begin{pmatrix} a \amp b \\ c \amp d\end{pmatrix}\) such that

\begin{equation*} \begin{pmatrix} 4 \amp 1 \\ 3 \amp 1\end{pmatrix}\begin{pmatrix} a \amp b \\ c \amp d\end{pmatrix}=\begin{pmatrix} 1 \amp 0 \\ 0 \amp 1\end{pmatrix}. \end{equation*}

Thus \(a,b,c,d\) must satisfy:

\begin{align*} 3a+c\amp=1\\ 3b+d\amp=0\\ 5a+2c\amp=0\\ 5b+2d\amp=1 \end{align*}

So solving this system:

Gives us \(a=2, b=-1, c=-5, d=3\text{.}\) Thus

\begin{equation*} M^{-1}=\begin{pmatrix} 2 \amp -1 \\ -5 \amp 3\end{pmatrix}. \end{equation*}

Subsection 2.4.2 Non-Invertible Matrices

Suppose we had

\begin{equation*} M=\begin{pmatrix} 1 \amp 1 \\ -1 \amp -1\end{pmatrix}. \end{equation*}

If we tried to find it's inverse as we have above, we would have:

\begin{equation*} \begin{pmatrix} 1 \amp 1 \\ -1 \amp -1\end{pmatrix}\begin{pmatrix} a \amp b \\ c \amp d\end{pmatrix}=\begin{pmatrix} 1 \amp 0 \\ 0 \amp 1\end{pmatrix}. \end{equation*}

and the system

\begin{align*} a-c\amp=1\\ b-d\amp=0\\ a-c\amp=0\\ b-d\amp=1 \end{align*}

But wait! How can \(a-c=1\) and \(a-c=0\text{?}\) This is a non-solvable system. We can see it here as well:

We see that we have a row \(0a+0b+0c+0d=1\text{,}\) which has no solution.

Remark 2.4.2.
Not every matrix is invertible, such matrices are called singular matrices.

Subsection 2.4.4 Solving Systems of Equations (again)

Let's say we wanted to solve:

\begin{align*} 0.3x+0.8y+0.5z\amp =10\\ 1.5x+0.2y+0.7z\amp =15\\ .5x-0.5y+0.2z\amp =20 \end{align*}

We can do this with reduced row echelon form, but note that we can rephrase this system as a single matrix equation:

\begin{equation*} \begin{pmatrix} 0.3 \amp 0.8 \amp 0.5 \\ 1.5 \amp 0.2 \amp 0.7\\ 0.5 \amp -0.5 \amp 0.2 \end{pmatrix} \begin{pmatrix}x\\y\\z\end{pmatrix}=\begin{pmatrix}10\\15\\20\end{pmatrix}. \end{equation*}

Thus, if our left hand matrix is invertible, we have:

\begin{equation*} \begin{pmatrix}x\\y\\z\end{pmatrix}=\begin{pmatrix} 0.3 \amp 0.8 \amp 0.5 \\ 1.5 \amp 0.2 \amp 0.7\\ 0.5 \amp -0.5 \amp 0.2 \end{pmatrix}^{-1}\begin{pmatrix}10\\15\\20\end{pmatrix}. \end{equation*}
