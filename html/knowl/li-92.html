Item.

If \(n=m\text{,}\) then \(\frac{a_n}{b_m}x^{n-m}=\frac{a_n}{b_m}\) is just a constant. So as the magnitude of \(x\) grows, these values approach \(\frac{a_n}{b_m}\text{.}\) For example consider \(y=\frac{-2x^4+x}{3x^4+1}\text{.}\) We should note: \(\frac{-2x^4-x}{3x^4+1}\sim\frac{-2x^4}{3x^4} \sim -\frac{2}{3}\text{,}\) so the long term behavior of \(y\) will be the same as \(-\frac{2}{3}\)'s: \url{https://www.desmos.com/calculator/qaeq86mivp}.
