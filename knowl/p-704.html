So we are solving:

\begin{align*} 4000\amp = 100\frac{1-(1.02)^{-n}}{.02}\\ 40 \amp = \frac{1-(1.02)^{-n}}{.02}\\ 0.8 \amp = 1-(1.02)^{-n}\\ (1.02)^{-n} \amp = 0.2\\ -n\ln((1.02)) \amp = \ln(0.2)\\ n\amp = \frac{\ln(0.2)}{-\ln(1.02)}\approx 81.274 \end{align*}

Or, by plotting \(y=\frac{1-(1.02)^{-x}}{.02}\) and \(y=40\text{,}\) we can see what the value of \(n\) should be: \(n\approx 81.274\text{.}\)
