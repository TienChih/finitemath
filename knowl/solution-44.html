So we note that:

\begin{align*} A\amp = 100000\\ P\amp = 50000\\ i\amp = 0.07 \end{align*}

If one is familiar with the properties of logs and natural logs:

\begin{align*} 100000\amp =50000(1.07)^{n}\\ 2\amp =(1.07)^{n}\\ \ln(2)\amp= \ln(1.07^n)\\ \ln(2)\amp= n\ln(1.07)\\ n\amp=\frac{\ln(2)}{\ln(1.07)} \approx 10.24477 \end{align*}

So about \(10.24477\) years.

Otherwise, since we know \(P, i\) we can simply plot the graph and see when it reaches 50000: So about 10.245 years.
