\begin{equation*} (2,1),(2,2),(2,3),(2,4),(2,5),(2,6) \end{equation*}
\begin{equation*} (3,1),(3,2),(3,3),(3,4),(3,5),(3,6) \end{equation*}
\begin{equation*} (4,1),(4,2),(4,3),(4,4),(4,5),(4,6) \end{equation*}
\begin{equation*} (5,1),(5,2),(5,3),(5,4),(5,5),(5,6) \end{equation*}
\begin{equation*} (6,1),(6,2),(6,3),(6,4),(6,5),(6,6) \end{equation*}
  1. Note that \(A= \{(6,1),(6,2),(6,3),(6,4),(6,5),(6,6)\}\) so \(P(A)=\frac{6}{36}.\text{.}\)
  2. Note that \(B= \{(4,6),(5,5),(5,6),(6,4),(6,5),(6,6)\}\) so \(P(B)=\frac{6}{36}\text{.}\)
  3. Note that \(C= \{(1,3),(2,3),(3,3),(4,3),(5,3),(6,3)\}\) so \(P(C)=\frac{6}{36}\text{.}\)
  4. Note that \(A\cap B= \{(6,4),(6,5),(6,6)\}\) so \(P(A\cap B)=\frac{3}{36}\text{.}\)
  5. \(P(A\cup B)=P(A)+P(B)-P(A\cap B)=\frac{6}{36}+\frac{6}{36}-\frac{3}{36}=\frac{9}{36}\text{.}\) Also \(A\cup B=\{(4,6),(5,5),(5,6), (6,1),(6,2),(6,3),(6,4),(6,5),(6,6)\}\) so \(P(A\cup B)=\frac{9}{36}\text{.}\)
  6. Note \(B\cap C=\emptyset\) so \(P(B\cap C)=0\text{.}\) On other words \(B, C\) are disjoint.
