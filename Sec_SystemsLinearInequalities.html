Section 3.2 Systems of Linear Inequalities

If a solution to a system of equations is a set of points that make several equations true, then the same is true of systems of linear inequalities. It's just going to be the collection of points that make all the inequalities true.

So if say we had the system of linear inequalities:

\begin{align*} x+y\amp \geq 4\\ x-2y\amp \leq 3 \end{align*}

We could ask ourselves if \((0,0), (4,4), (6,0), (2,-2)\) belong to the region.

  1. For \((0,0)\) note that:
    \begin{align*} 0+0\amp \not\geq 4\\ 0-2(0)\amp \leq 3 \end{align*}
    So we satisfy the second but not the first inequality, so \((0,0)\) does not belong to the region.
  2. For \((4,4)\) note that:
    \begin{align*} 4+4\amp \geq 4\\ 4-2(4)\amp \leq 3 \end{align*}
    So we satisfy both inequalities, and \((4,4)\) does belong to the region!
  3. For \((6,0)\) note that:
    \begin{align*} 6+0\amp \geq 4\\ 6-2(0)\amp \not\leq 3 \end{align*}
    So we satisfy the first inequality but not the second, so \((6,0)\) does not belong to the region.
  4. For \((-2,2)\) note that:
    \begin{align*} 2+(-2+\amp \not\geq 4\\ 2-2(-2)\amp \not\leq 3 \end{align*}
    So we satisfy neither inequality but not the second, and \((2,-2)\) does not belong to the region.

We can use the Desmos graph and drag the point \((a,b)\) around to verify which region satisfies both inequalities.

Example 3.2.1.

What are the solutions to the following system:

\begin{align*} 2x+3y\amp \leq 12\\ x\amp \geq 0\\ y\amp \geq 0 \end{align*}

We know from earlier that the solutions have to lie in the bottom right hand of the plane on or below the line \(2x+3y=12\text{.}\) We also know that neither the \(x\) or \(y\) values can be negative. What's left is a sort of triangle: This is the overlap of regions where all 3 inequalities are true.
