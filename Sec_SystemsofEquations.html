Section 2.1 Systems of Linear Equations

A system of equations with unknowns \(x_1,\ldots,x_n\) is a collection of first degree equations:

\begin{align*} a_{1,1}x_1+a_{1,2}x_2+\cdots+a_{1,n}x_n\amp = b_1\\ a_{2,1}x_1+a_{2,2}x_2+\cdots+a_{2,n}x_n\amp = b_1\\ \vdots \amp = \vdots\\ a_{m,1}x_1+a_{m,2}x_2+\cdots+a_{m,n}x_n\amp = b_1 \end{align*}

Subsection 2.1.1 What Exactly is a Solution to a System of Linear Equations?

To answer the question that is the name of this section, we should kinda back up here and ask “What is a solution to an equation?”

For example, what is a solution to \(y=2x+3\text{?}\) After dealing with linear equations in Chapter 1, we should have some sense as to what \(y=2x+3\) looks like, that is a line with slope 2 and \(y\)-intercept \((0,3)\text{.}\) But what IS this line? It's the collection of all points \((x,y)\) that make the above equation true. So \((0,3), (-1,1)\) and \((2,7)\) are all on this line because they make the equation true (\(3=2(0)+3, 1=2(-1)+3, 7=2(2)+3\)). A point like \((1,1)\) is NOT on this line because \(1\neq 2(1)+3\text{.}\)

So a solution to an equation is any point which makes the equation true.

A System of Equations is a collection of equations. Thus, a solution to a system of equations is any point which makes all the equations true.

Subsection 2.1.2 What are the Types of Solutions that we can have?

So a solution to the system:

\begin{align*} 2x+3y\amp = 15\\ x-y\amp = 0 \end{align*}

is a collection of points that satisfy the first AND the second linear equalities. The first equation describes the line: \(y=-\frac{2}{3}x+5\text{,}\) whereas the second describes the line \(y=x\text{.}\) Each of these equations represent lines, so each individually has infinitely many solutions. The question is, how many solutions do these lines have in common?

There are a number of ways to find out, but the most straight forward is simply to graph the system:

As one can see, although each line has infinitely many points, the only solution they have in common is the point \((3,3)\text{.}\) To verify this, we note that \(2(3)+3(3)=15\) and \(3-3=0\text{,}\) so both equations are satisfied. This system has a unique solution

What else could happen when solving a system of equations? Well, consider:

\begin{align*} x+y\amp = 5\\ 2x+2y\amp = 10\text{.} \end{align*}

If we look at the graph of both lines, we only see one actual line:

This is because both equations describe the same line. After all, whenever \(x+y=5\text{,}\) it would have to be true that \(2(x+y)=5\cdot2\) and thus \(2x+2y=10\text{.}\) So ANY point that is a solution to one equation is a solution to the other, this system has infinitely many solutions.

Another possibility arises from the following example:

\begin{align*} x+y\amp = 5\\ x+y\amp = 6 \end{align*}

If we graph these lines side by side:

We can see that these lines are parallel and do not actually have any points in common. After all, how many pairs of numbers \(x\) and \(y\) sum to both 5 AND 6? It should be clear that this system has no solutions.

Of course, these have all been Systems of Linear Equations in 2 variables. When we increase the number of variables, we increase the dimensionality of the potential solution space, and what it means to be “linear” changes somewhat. Nevertheless, these examples illustrate the 3 possible outcomes of a system of linear equations in any dimension:

  • There is a unique Solution.

  • There are infinitely many Solutions.

  • There are no Solutions.

Subsection 2.1.3 How do we find the Solution(s)/know we have No Solutions?

In 2 variables, it's simple enough we can just graph, as we did above. But we should try to develop some algebraic techniques as well, since this will be harder to do when we increase the dimensions of our problems.

What we introduce here is the Echelon Method, the general goal is to take non-zero multiples of our equations and use them to eliminate variables in the other equations so that a solution to a variable can be made known. This process should also reveal if there are infinitely many or no solutions.

Example 2.1.1.
\begin{align*} 2x+3y\amp = 1\\ x+y\amp = 0 \end{align*}

We note that whatever the solutions to this are, they should be the same as the solutions to:

\begin{align*} x+\frac{3}{2}y\amp = \frac{1}{2}\\ x+y\amp = 0 \end{align*}

We've divided the first equation in half, but we haven't actually changed what points will make that equation true. Then:

\begin{align*} x+\frac{3}{2}y\amp = \frac{1}{2}\\ x+y-(x+\frac{3}{2}y)\amp = 0-\frac{1}{2} \end{align*}

Must also be true, since we can add and subtract the same values from 2 sides of an equation and preserve the solutions. By doing this, we kill off the \(x\) in the second equation and we can solve for \(y\text{:}\)

\begin{align*} x+\frac{3}{2}y\amp = \frac{1}{2}\\ -\frac{1}{2}y\amp = -\frac{1}{2} \end{align*}

So \(y=1\) based on the second line.

\begin{align*} x+\frac{3}{2}y\amp = \frac{1}{2}\\ y\amp = 1 \end{align*}

If we take \(-\frac{3}{2}\) times the second line and add it to the first:

\begin{align*} x+\frac{3}{2}y-\frac{3}{2}(y)\amp = \frac{1}{2}-\frac{3}{2}(1)\\ y\amp = 1 \end{align*}

We obtain:

\begin{align*} x\amp = -1\\ y\amp = 1 \end{align*}

We can verify that if we consider the original system:

\begin{align*} 2x+3y\amp = 1\\ x+y\amp = 0 \end{align*}

then by evaluating at \(x=-1, y=1\text{:}\)

\begin{align*} 2(-1)+3(1)=-2+3\amp = 1\\ -1+1\amp = 0 \end{align*}
Example 2.1.2.

So if we consider:

\begin{align*} x+y\amp = 2\\ 2x+2y\amp = 4\text{.} \end{align*}

Just like before, we want to kill the \(x\) in the second equation:

\begin{align*} x+y\amp = 2\\ 2x+2y-2(x+y)\amp = 4-2(2)\text{.} \end{align*}

Which gives us:

\begin{align*} x+y\amp = 2\\ 0\amp = 0\text{.} \end{align*}

Well..., certainly that second line doesn't contribute much, zero is always equal to itself. So the solutions are just whenever \(x+y=2\text{,}\) which happens for infinitely many possible pairs.

Example 2.1.3.

Lastly, when we look at:

\begin{align*} x+y\amp = 0\\ x+y\amp = 1\text{.} \end{align*}

We can try to kill the \(x\) in the second line as before:

\begin{align*} x+y\amp = 0\\ x+y-(x+y)\amp = 1-0\text{.} \end{align*}

Where we get:

\begin{align*} x+y\amp = 0\\ 0\amp = 1\text{.} \end{align*}

So all you have to do is find a pair \((x,y)\) so that \(0=1\text{...}\) which is not ever going to happen. So this will never have a solution.
